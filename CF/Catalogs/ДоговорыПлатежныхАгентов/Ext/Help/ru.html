Договоры платежных агентов

Справочник предназначен для хранения информации обо всех договорах, заключенных между предприятием (собственными организациями) и поставщиками услуг (или платежными агентами - в случае субагентских договоров) по приему платежей от физических лиц.

Список договоров открывается по ссылке Финансы – См. также – Агентские договоры с поставщиками. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

 

Новую запись в справочник можно добавить с помощью кнопки Создать (Клавиша Insert). в командной панели списка. Необходимо заполнить поля:

 

Признак агента - собственное юридическое лицо, заключившее договор:

· Банковский платежный агент

· Банковский платежный субагент

· Платежный агент

· Платежный субагент

· Поверенный

· Комиссионер

· Агент

· Комитент

Становится доступной вкладка "Комиссионная торговля", заполняются реквизиты:

· "Услуга" тип справочник "Номенклатура" с типом номенклатуры "Услуга"

· "Способ расчета", перечисление "Способы расчета вознаграждения"

· "Процент" доступность элемента зависит от значения реквизита "Способ расчета"

· Заполняются реквизиты учета НДС комитента, реквизит "ИНН" у контрагента должен быть заполнен

Организация - собственное юридическое лицо, заключившее договор.
Банковский счет организации - денежные средства платежных агентов и субагентов, как правило, учитываются на отдельном расчетом счете организации.

Агент - указывается контрагент, который предоставляет услуги.

Указание посредника - Посредник указывается в случае, если договор является субагентским. В противном случае реквизит не отображается в форме.

Вознаграждение - если при оплате услуг платежным агентом удерживается комиссия, то указывается процент комиссионного вознаграждения.

Для записи элемента справочника используется команда "Записать и закрыть" (Сочетание клавиш Ctrl + Enter) или "Записать" (Сочетание клавиш Ctrl + s).
