Форматы магазинов

Формат магазина – это тип магазина, который характеризуется его местоположением (магазин у дома, районный супермаркет), торговой площадью (гипермаркет, супермаркет, минимаркет), отраслью (супермаркет, универмаг), ценовой категорией (бутик, outlet, дискаунтер), широтой и глубиной представленного ассортимента (специалист в категории, дискаунтер).

Список элементов открывается по ссылке Маркетинг – Ассортимент – Форматы магазинов, (если выставлен флажок Управление ассортиментом по ссылке Администрирование-Маркетинг-Ассортимент). С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка. В программе предоставляется возможность внести описание формата магазина и целевой группы потребителей. Каждому формату можно назначать следующие параметры.

Учетные параметры устанавливаются для дальнейшего использования по умолчанию и проведения итогов работы менеджера:

· Правило ценообразования – которое будет использоваться для определения вида цен при первоначальном вводе номенклатуры в ассортимент;

· Количество ассортиментных позиций – то есть количество разновидностей или марок товаров в магазине;

Прочие параметры несут описательный характер и не используются для контроля действий пользователя:

· Минимальная площадь торгового зала – при которой магазин может быть включен в данный формат;

· Максимальная площадь торгового зала – при которой магазин может быть включен в данный формат;

· Технология размещения товара и обслуживания покупателей – способы укладки и представления покупателям товаров. Например, магазин самообслуживания или магазин, продающий товары по каталогам;

· Целевое местоположение – наиболее целесообразное географическое положение магазина. Например, магазины автозапчастей есть смысл располагать рядом с магистралями; книжные, специализирующиеся на литературе по музыке, – рядом с музыкальным училищем и т. п.;

· Целевые группы покупателей – группы покупателей, разделенных по социальным и/или демографическим характеристикам (пол, возраст, профессия и др.), способных и готовых приобретать товар в магазине данного формата, и на продажу которым нацелена маркетинговая политика компании. Например, целевой группой детского магазина будут семьи с детьми от 0 до 13 лет, магазина шаговой доступности – жители окрестных домов.

Значения числовых параметров вводятся в соответствующие поля, а значения текстовых можно выбирать в выпадающем списке из уже введенных в системе либо назначать новые. После сохранения документа вновь введенное значение параметра становится доступным в других документах.
