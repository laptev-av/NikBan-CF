Единицы измерения

В справочник «Единицы измерения» вводится список используемых в магазине основных единиц измерения (штуки, килограммы, метры и т.п.). Справочник имеет одноуровневую структуру.

Список элементов открывается по ссылке НСИ – Номенклатура –Единицы измерения. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы, задавая поиск по значениям различных реквизитов.

Новый элемент можно создать вручную или воспользоваться общероссийским классификатором единиц измерения (ОКЕИ).

Для добавления данных из общероссийского классификатора необходимо нажать на кнопку Подобрать из классификатора. Откроется таблица общероссийского классификатора единиц измерения. Уже используемые единицы измерения в классификаторе подсвечены серым. Пользователь может отметить нужные единицы измерения флажком, и при нажатии кнопки ОК они будут добавлены в список единиц измерения.
