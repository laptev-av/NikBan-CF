Группы получателей скидки

Справочник Группы получателей скидки предназначен для создания групп держателей дисконтных карт и используется при назначении скидок. Данные о получателях хранятся в регистре сведений "Состав групп получателей скидки".

Список групп открывается по ссылке Маркетинг – Маркетинговые акции – Группы получателей скидки. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные группы, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

После сохранения записи перейдите к форме редактирования настроек схемы компоновки данных – команда Настройки.

Состав сегмента формируется или вручную, или периодически обновляется на основе СКД аналогично сегментам номенклатуры. Для формирования сегмента можно использовать ответы, полученные при опросе покупателей. На скриншоте приведен пример настройки сегмента: периодическое пополнение списка покупателей-мужчин, имеющих личный транспорт.

Покупатель сможет пользоваться всеми скидками, назначенными для данной группы получателей скидки.
