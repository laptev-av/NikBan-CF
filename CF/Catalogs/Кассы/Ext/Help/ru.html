Кассы

В справочник Кассы вводится информация об операционных кассах предприятия. Операционные кассы предназначены для учета движения денежных средств между предприятием и внешними контрагентами (поставщиками, банками). Регистрируются также перемещения денежных средств между кассами ККМ и операционной кассой внутри магазина и между операционными кассами разных магазинов.

Список всех операционных касс предприятия можно открыть по ссылке НСИ - Предприятие - Кассы. Кроме того, по ссылке Кассы из панели навигации карточки магазина (организации) можно открыть список касс данного магазина (организации). Для удобства работы Пользователь может отфильтровать список касс по магазину, выбрав нужное значение в поле Магазин в шапке списка. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные кассы, задавая поиск по значениям различных реквизитов.

Новую операционную кассу можно ввести, нажав кнопку Создать в списке касс.

 

В открывшейся форме нужно ввести магазин, организацию, к которой относится касса, и наименование кассы. Если предполагается печать документа на узколенточном принтере, можно выбрать предварительно настроенный шаблон печати кассовых ордеров.

Для учета чеков, пробитых из документов ПКО в справочник Кассы есть возможность настройки кассы ККМ, по которой пробивались чеки. Для этого на форме элемента справочника Кассы нужно установить флажок Учитывать ДС в кассе ККМ и выбрать нужную кассу из справочника Кассы ККМ. При этом список касс ККМ фильтруется по организации, магазину и типу кассы - доступны кассы типа "Фискальные регистраторы или АСПД" и "ККМ без подключения оборудования". Денежные средства при приеме денежных средств от покупателя или выдаче при возврате будут учтены в кассе ККМ до момента закрытия смены и их выемки в кассу общей суммой.
