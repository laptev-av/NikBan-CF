Справочник виды алкогольной продукции

В конфигурации поддерживается формирование журнала учета продаж алкогольной продукции. Для формирования журнала необходимо настроить справочники Виды алкогольной продукции и Виды номенклатуры, а также определить для каждой номенклатуры параметры учета. В программе предоставляется возможность хранить информацию о лицензиях поставщиков на оптовую продажу спиртосодержащей продукции. Функциональность доступна при установке функциональной опции Алкогольная продукция (Администрирование - Настройки номенклатуры).

Список элементов открывается по ссылке Нормативно-справочная информация - См. также- Виды алкогольной продукции.

С помощью кнопки Создать Пользователь может добавить новую запись в справочник Виды алкогольной продукции.

Значение вида алкогольной продукции и страны происхождения, подставляемые по умолчанию при создании номенклатуры можно указать в Виде номенклатуры. Для этого необходимо установить флаг Алкогольная продукция для данного вида номенклатуры. Это может быть удобно, если торговое предприятие преимущественно продает какой-то определенный вид алкогольной продукции, например, в пивном баре.
