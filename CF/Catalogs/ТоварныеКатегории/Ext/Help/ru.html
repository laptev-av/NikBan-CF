Товарные категории

Товарная категория – это совокупность товаров, которые покупатель воспринимает как схожие между собой, или товаров совместного использования. Более простое и понятное определение товарной категории – это то, за чем (за каким товаром) покупатель идет в магазин. Товарная категория может быть представлена марочным товаром или товаром, не имеющим собственного бренда.

Список товарных категорий открывается по ссылке Маркетинг – Ассортимент – Товарные категории, (если выставлен флажок Квоты ассортимента и товарные категории по ссылке Администрирование-Маркетинг-Ассортимент). В левой части окна редактирования товарных категорий отображается справочник Виды номенклатуры. В данной форме он доступен только для просмотра, но не для редактирования. Для каждого вида номенклатуры, доступного в системе, можно создать иерархию товарных категорий.

Раскройте какой-либо вид номенклатуры и выберите элемент, справа раскроется товарные категории. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные товарные категории, задавая поиск по значениям различных реквизитов. Новая группа товарных категорий создается с помощью кнопки Создать новую группу.

Новую товарную категорию в справочник можно добавить с помощью кнопки Создать в командной панели списка. При создании товарной категории необходимо ввести Наименование и, по необходимости группу категорий верхнего уровня. Вид номенклатуры при создании заполняется автоматически выделенной позицией справочника Виды номенклатуры в левом окне.
