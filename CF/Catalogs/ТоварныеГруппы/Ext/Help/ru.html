Товарные группы

Справочник Номенклатурные группы используется для группировки товаров по местам продажи в торговом зале магазина. Группы могут создаваться по любому принципу, например, по виду торгового оборудования, на котором будет размещаться товар, или по принципу близости потребительских свойств.

Принадлежность товара к определенной номенклатурной группе используется при автоматическом распределении поступления и отгрузки товаров по торговым залам или отделам магазина.

Список групп открывается по ссылке НСИ – Номенклатура – Товарные группы. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные группы, задавая поиск по значениям различных реквизитов. С помощью кнопки Создать новую группу, пользователь может создать иерархическую структуру групп.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.
