Статьи движения денежных средств

Справочник используется для детализации учета потока денежных средств. Чаще всего в более строгом контроле нуждается затратная часть расходов.

Список элементов открывается по ссылке Администрирование – Организации и финансы - Финансы – Статьи движения денежных средств. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы, задавая поиск по значениям различных реквизитов. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов.

Пример. Для хозяйственной операции «Оплата поставщику» политикой предприятия в части учета денежных средств предусмотрены еще три статьи:
- Оплата поставщикам товара
- Оплата поставщикам за расходные материалы торгового процесса
- Оплата поставщикам за канцелярские товары
Таким образом, проанализировав отчет о движении денежных средств за период можно сделать вывод направлении расходования денежных средств.

Новую запись в справочник (новый аналитический признак) можно добавить с помощью кнопки Создать в командной панели списка.

В открывшейся форме ввести наименование аналитики и указать из перечисления хозяйственную операцию, к которой она относится.
