Договоры эквайринга

Договор эквайринга предназначен для регистрации видов платежных карт и условий оплаты услуг банка-эквайрера.

Список договоров открывается по ссылке Финансы – См. также – Договоры эквайринга (Администрирование – Продажи – Договоры эквайринга). С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные договора, задавая поиск по значениям различных реквизитов.

Пользователь может установить быстрый отбор по организации предприятия, чтобы просмотреть заключенные от имени этой организации договоры.

Новый договор можно добавить с помощью кнопки Создать в командной панели списка.

В открывшейся форме нужно выбрать организацию предприятия и контрагента – банк, между которыми заключается договор. Наименование договора формируется автоматически и может быть изменено пользователем. Флажок Эквайрер возвращает комиссию при возврате позволяет учесть ситуацию, когда возврат денежных средств на платежную карту покупателя в течение кассовой смены не влечет за собой начисления комиссии.

В табличной части документа пользователь может выбрать один или несколько видов оплат, принимаемых банком, и соответствующий виду оплат процент торговой уступки (комиссии) за проведение платежей.

В Договоре эквайринга также перечисляются эквайринговые терминалы, с помощью которых производится оплата платежной картой. Список терминалов можно открыть из меню Эквайринговые терминалы панели навигации формы договора.

Платежные терминалы могут подключаться к программе, т.е. работать с автоматической передачей данных в платежную систему, или работать без подключения. В последнем случае терминал должен обладать функцией печати слип-чека на встроенном печатающем устройстве.

При создании новой записи справочника Эквайринговый терминал пользователь может указать тип кассы и выбрать нужную кассу из предложенного списка. По типу кассы "Кассы ККМ" будут учитываться оплаты, произведенные с помощью документов "Чек ККМ", по типу "Кассы" - "Эквайринговая операция".

Если терминал платежной системы работает в режиме подключения к программе, то необходимо выбрать соответствующий ему элемент справочника Подключаемое оборудование, в противном случае - установить флажок Использовать ЭТ без подключения к системе.
