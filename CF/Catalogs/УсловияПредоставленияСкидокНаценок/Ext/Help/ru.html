Условие предоставления скидок (наценок, ограничений)

Система поддерживает следующие условия предоставления скидок:

· За разовый объем продаж – продажи, оформленные одним документом;

· За накопленный объем продаж – продажи за определенный период;

· За время продаж – продажи, совершенные в определенный интервал времени;

· За комплект покупки – за определенные товары, купленные одновременно;

· В день рождения – за покупку, совершенную в определенный интервал времени, включающий день рождения покупателя;

· Порядковый номер продажи кратен (n) – определенный вид лотереи, когда скидка предоставляется каждому n-му покупателю. Количество продаж – это количество оформленных чеков продаж, без учета отложенных, аннулированных и чеков возврата;

· По типу покупателя – скидка определенной группе покупателей.

Список элементов открывается по ссылке Маркетинг – Маркетинговые акции - Условие предоставления скидок (наценок, ограничений). С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы, задавая поиск по значениям различных реквизитов.

Новую запись в справочник добавить с помощью кнопки Создать в командной панели списка. В каждой скидке может быть указано несколько условий, которые для применения скидки должны выполниться одновременно.

Если в условии указан определенный сегмент номенклатуры, то оно будет проверяться только по товарам, входящим в этот сегмент.

Наименование условия скидки формируется автоматически исходя из введенных параметров условий скидки.

За разовый объем продаж

В списке условий предоставления скидки выберите За разовый объем продаж и заполните условия расчета объема продаж. Скидка может рассчитываться по Сумме или Количеству вошедших в документ товарных позиций с ограничением Не более или Не менее введенного пользователем числа. Если указан определенный Сегмент номенклатуры, то условия будут проверяться только по товарам, входящим в сегмент.

Наименование условия формируется автоматически исходя из введенных параметров условий скидки.

За накопленный объем продаж

В списке условий предоставления скидки выберите За накопленный объем продаж и заполните условия расчета условий. Накопления рассчитываются по дисконтной карте покупателя.

Накопления рассчитываются в виде Суммы или Количества проданного по дисконтной карте товара и могут быть ограничены - не более или не менее введенного пользователем значения. Накопления могут подсчитываться как за Весь период покупок или за период, кратный указанному значению:год, полугодие, квартал, месяц. Объем продаж может быть рассчитан С начала текущего периода времени, за Прошлый период, за Прошлый скользящий период и за Предыдущую покупку.

Если указан определенный Сегмент номенклатуры, то условия будут проверяться только по товарам, входящим в сегмент.

Наименование условия формируется автоматически исходя из введенных параметров условий скидки.

За время продаж

В списке условий предоставления скидки выберите За время продажи продаж и заполните Время начала и Время окончания для каждого Дня недели действия скидки.

Наименование условия формируется автоматически исходя из введенных параметров условий скидки.

За комплект покупки

Пользователь может указать в качестве условий скидки перечень товаров, которые для получения скидки должны быть куплены одновременно. Например, при покупке садового инвентаря, 2-х лопат и грабель, секатор предоставляется в виде скидки.

Выберите значение условия скидки За комплект покупки и добавьте в список Номенклатуру, указав по необходимости Упаковку и Количество упаковок для каждого товара, входящего в комплект.

В день рождения

Выберите значение условия скидки В День рождения покупателя и введите период предоставления скидки. Если дата рождения покупателя введена в его анкетные данные, то скидка будет предоставляться автоматически в период за указанное количество Дней до Дня рождения покупателя и в течение указанного количества Дней после Дня рождения.

Если информация о дате рождения не внесена в базу данных, то скидку по предъявлении паспорта можно настроить в виде ручной скидки, предоставляемой кассиром при предъявлении паспорта.

Кратность номера чека

Чтобы скидка предоставлялась по номеру чека, нужно выбрать условия предоставления скидки Кратность номера чека и указать в параметрах нужное число. Скидка будет предоставлена, если номер чека делится на указанное число без остатка. Например, на пять.

По типу покупателя

Выберите в условиях предоставления скидки условие По типу покупателя. Система предоставляет возможность сформировать список получателей скидок по различным признакам:

· Вид дисконтной карты - скидка будет предоставлена покупателям, обладающим дисконтной картой определенного вида;

· Дисконтная карта - скидка будет предоставлена обладателям перечисленных в списке дисконтных карт;

· Физическое лицо - скидка будет предоставлена перечисленным в списке физическим лицам;

· Контрагент - скидка предоставляется перечисленным контрагентам;

· Группа получателей скидки - скидка будет предоставлена всем, входящим в перечисленные группы получателей скидки.

В название скидки автоматически добавляется тип получателя скидки. Пользователь может вручную добавить краткое описание получателей скидки данного типа, чтобы различать похожие условия в списке.
