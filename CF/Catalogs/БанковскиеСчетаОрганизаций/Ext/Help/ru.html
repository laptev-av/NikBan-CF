Банковский счет организации

Банковские счета предназначены для хранения сведений о банковских счетах юридических и физических лиц, как собственных, так и сторонних.

Список счетов организации открывается по ссылке НСИ – Организации – выберите, затем откройте карточку контрагента – Еще – Перейти - Банковские счета. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные счета, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

В открывшейся форме введите номер счета и БИК банка. Корсчет, Наименование банка и Город, в котором расположен филиал банка определится автоматически по данным классификатора Банки. Пользователь может выбрать наименование для созданного банковского счета организации из числа автоматически созданных системой или ввести свое.
