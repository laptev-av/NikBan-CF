Марки (Бренды)

Марка (бренд) – это зарегистрированный знак или наименование, по которому покупатель идентифицирует товар, отличая его от другого аналогичного. На рынке, как правило, представлены бренды производителей продукции или собственные марки торговых сетей.

Список элементов открывается по ссылке НСИ (Номенклатура – Марки (Бренды). С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные марки, задавая поиск по значениям различных реквизитов. С помощью команды Создать группу пользователь может создать новый уровень иерархии в дереве марок.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.
