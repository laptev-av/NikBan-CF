Эквайринговые терминалы

Это устройство, подключенное по каналам связи к процессинговому центру банка-эквайрера и осуществляющее в реальном времени операции безналичных оплат по платежным картам. Устройство оснащено считывающим устройством магнитных карт и/или микросхемой. Совместная работа ККМ и терминала возможна как в режиме РМК, так и при оформлении документов Чек ККМ.

Список эквайринговых терминалов размещен в разделе Администрирование – Продажи – Оплата - Эквайринговые терминалы. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

При создании новой записи необходимо выбрать банк, предоставляющий услуги эквайринга в поле Эквайрер, из списка контрагентов и ввести наименование Организации предприятия, с которой заключен договор эквайринга. Затем необходимо выбрать кассу предприятия, по которой будут учитываться принятые с помощью настраиваемого терминала платежи, и выбрать подключенный к данному рабочему месту терминал из списка подключенного оборудования.

Если платежи будут проводиться без подключения терминала к компьютеру пользователя, то следует установить флажок Использовать без подключения оборудования. В противном случае для каждого терминала следует предварительно создать запись в справочнике Подключаемое оборудование.

Параметры договора вводятся на следующей закладке -. Для каждого из видов оплат, (Доступные виды оплаты), принимаемых на данном терминале, можно указать размер комиссии, начисляемой банком при продаже. А также указать, каким образом взимается комиссия при отмене платежа в течение смены и при возврате товара по окончании смены продажи.

Объем оплат, произведенных покупателями с помощью платежных карт можно просмотреть с помощью отчета Продажи по платежным картам раздела Финансы – Отчеты по финансам.
