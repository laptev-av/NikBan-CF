Шаблоны работы дисплея покупателя

Пользователь может настроить шаблон вывода информации на дисплей покупателя для определенной настройки рабочего места кассира из справочника Настройки РМК. На закладке Общие настройки можно выбрать шаблон из списка предварительно созданных шаблонов, настроенных для разных моделей, используемых на предприятии дисплеев покупателей.

Список шаблонов открывается по ссылке Администрирование – Подключаемое оборудование - Шаблоны работы дисплея покупателя. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

Для каждого шаблона можно указать:

· Текст при простое кассы, выводимый между оформлениями чеков в кассах, готовых к обслуживанию покупателей;

· Текст в нерабочем состоянии, выводимый при блокировке кассового аппарата для сервисного обслуживания;

· Выводить промежуточный итог чека – флажок, указывающий, нужно ли выводить подитог чека при сканировании каждой позиции.

Если шаблон работы дисплея покупателей не выбран, то выводится информация, установленная в конфигурации «Розница» по умолчанию.

Работа с дисплеем покупателя

При оформлении документов Чек ККМ на дисплее покупателя отображается информация по текущей строке чека: наименование товарной позиции, количество и цена товарной позиции, скидка и итоговая сумма по строке. Если в шаблоне включен режим вывода предварительного итога, то во второй строке дисплея отображается текущая сумма чека с учетом скидки.

При переходе к оплате на дисплей выводятся последовательно:итоговая сумма чека к оплате, полученная от покупателя сумма и рассчитанная сумма сдачи, в том числе и нулевая. При оформлении возврата отображается аналогичная информация. Информация о скидке, примененной при продаже, не отображается, выводится только итоговая цена и сумма по строке.

При простое кассы отображается текст из предварительно настроенного шаблона работы с дисплеем покупателя. Режим простоя кассы включается после оформления оплаты чека, откладывания или аннулирования чека через 30 секунд при условии, что формирование нового чека еще не начато.
