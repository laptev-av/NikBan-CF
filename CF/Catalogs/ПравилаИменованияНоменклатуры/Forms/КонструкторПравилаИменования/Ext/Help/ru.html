Конструктор правила именования

Конструктор, с помощью которого создаются правила именования позиций номенклатуры. Расположен: НСИ - Номенклатура - Правила именования номенклатуры и доступен если установлен флаг Правила именования номенклатуры по ссылке Администрирование – Настройки номенклатуры – Общие параметры.

Форма содержит список параметров именования: реквизиты справочника "Номенклатура", дополнительные реквизиты и сведения справочника "Номенклатура". С помощью этой формы пользователь заполняет правило, по которому будут именоваться позиции номенклатуры.

Новое правило можно создать из карточки «Вид номенклатуры», для этого необходимо нажать F8 в поле «Правило именования». Список параметров, которые могут быть отобраны в правило:

  • Реквизиты справочника Номенклатура;
  • Общие свойства справочника Номенклатура. Видны при включенной опции Дополнительные реквизиты и сведения (Администрирование – Общие настройки). Путь к данным заполняется значением поля имя для разработчиков (Администрирование – Общие настройки – Дополнительные реквизиты – Номенклатура – Выбрать необходимый элемент номенклатуры - Выбрать Дополнительный реквизит – Для разработчиков);
  • Свойства справочника Номенклатура по видам номенклатуры. Видны при включенной опции Дополнительные реквизиты и сведения. Путь к данным заполняется значением поля имя для разработчиков;

Если в некоторых строках параметров не указан путь в поле Путь к данным, то такие параметры нельзя выбрать в правило. В этом случае в верхней части формы будет видна ссылка Указать путь к данным. По нажатию на данную ссылку путь к данным сформируется автоматически, список параметров перезаполнится, параметры будут доступны для выбора в правило.

Правило составления наименования

Поле для формирования правила. Его можно заполнять параметрами именования, либо произвольным текстом. Ввод параметров в данное поле возможен перетаскиванием из реквизитов и сведений, либо вручную. Причем, параметры правила должны быть выделены квадратными скобками, например [ПутьКДаннымПараметраПример]. Путь к данным параметра должен состоять из одного слова, начинаться с буквы и не содержать символов ":;!@#$%^&-~`',?{}[]+=*/|\ ()""".

При формировании наименования будет оставаться без изменений тот текст правила, который не выделен квадратными скобками. Параметры правила будут заменены на соответствующие значения позиции номенклатуры. Если параметр типа булево и его значение равно ложь, в наименование будет подставлена пустая строка. Если параметр типа булево и его значение равно истина, в наименование будет подставлен сам параметр.

Проверка применения правила именования

Для проверки применения правила именования, необходимо заполнить поле Номенклатура. При этом система сформирует наименования по правилу и отобразит его в соответствующем поле.

С помощью кнопки Проверить правило происходит проверка заполнения Правила составления именования. Система проверяет:

· Расстановку квадратных скобок;

· Корректность параметров, введенных в квадратных скобках;

· Параметр, введенный в квадратных скобках, должен быть представлен в дереве реквизитов и сведений;

· Дополнительные реквизиты и сведения, которые выбраны в качестве параметров именования, должны принадлежать одному набору свойств, либо быть общими;

· В правило составления наименования введен хотя бы один параметр;

· Параметр может входить в правило один раз;

 

По кнопке Сохранить и закрыть происходит закрытие формы конструктора и заполнение формы элемента справочника Правила именования номенклатуры.
