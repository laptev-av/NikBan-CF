Справочник лицензии поставщиков алкогольной продукции

Сведения о лицензиях поставщиков на ведение оптовой торговли алкоголем хранятся в справочнике Лицензии поставщиков алкогольной продукции.

Список элементов открывается по ссылке НСИ - См. также- Лицензии поставщиков алкогольной продукции.

Для удобства работы Пользователь может отфильтровать список лицензий по контрагенту выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

С помощью кнопки Создать Пользователь может добавить новую лицензию.
