Контрагенты

Справочник Контрагенты предназначен для введения и хранения информации о юридических лицах и частных предпринимателях, с которыми предприятие контактирует, осуществляя свою деятельность.

Иерархию справочника можно формировать так, как удобно для работы предприятия. Например, можно выделить следующие группы контрагентов:

· Поставщики - контрагенты, которые поставляют свой товар непосредственно в магазины.

· Покупатели - осуществляют закупки мелким оптом непосредственно в магазинах.

· Банки - предоставляют услуги эквайринга в магазинах торговой сети.

Иерархия может быть многоуровневой. Например, внутри группы Поставщики можно выделить подгруппы поставщиков определенных видов товара или поставщиков с определенными условиями поставки.

Список контрагентов можно открыть по ссылке НСИ – Контрагенты. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов.  С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы справочника, задавая поиск по значениям различных реквизитов.

 

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

Заполнение карточки контрагента

В карточке Контрагента фиксируется необходимая для работы информация. Вводится наименование контрагента, как правило, соответствующее краткому наименованию его организации и указывается его правовой статус:

· юридическое лицо,

· физическое лицо,

· юридическое лицо за пределами РФ,

· индивидуальный предприниматель (ПБОЮЛ).

Для юридических лиц можно указать полное наименование организации контрагента и ввести ИНН, КПП и Код по ОКПО, если юридическое лицо зарегистрировано на территории Российской Федерации.

Для физических лиц указывается фамилия, имя и отчество физического лица, ИНН, Документ, удостоверяющий его личность и Код ОКПО, если физическое лицо зарегистрировано в качестве Индивидуального предпринимателя.

Также необходимо выбрать Система налогоблажения из выпадающего списка:

· Общая система налогооблажения

· Упрощенная система налогооблажения

В карточку можно также внести контактную информацию в соответствии с настройками видов контактной информации для контрагентов.
Чтобы ввести или изменить значения дополнительных сведений для контрагента, нужно нажать кнопку Открыть дополнительные сведения в командной панели списка магазинов или в командной панели карточки выбранного контрагента.
Состав контактной информации и дополнительных сведений может редактировать Администратор на форме Общие сведения раздела Администрирование.

Переходы к связной информации

Из панели навигации карточки контрагента можно открыть связанные с данным контрагентом следующие данные:

· Банковские счета, сведения о которых будут использоваться при выписке документов.

· Дисконтные карты, выданные контрагенту.

· Классификатор организаций ЕГАИС

· Лицензии поставщиков алкогольной продукции

· Поставляемая поставщиком номенклатура.

· Поставщики номенклатуры

В справочник пользователь может установить график оплаты для заполнения этапов оплат по умолчанию. Если у контрагента не выбран ни один график оплаты, то заполнение будет выполнено по предустановленному варианту - полная оплата наличными в день поставки.
