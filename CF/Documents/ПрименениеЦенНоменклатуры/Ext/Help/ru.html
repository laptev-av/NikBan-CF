Применение цен номенклатуры

Документ Применение цен номенклатуры вводит в действие цены, по которым магазин осуществляет продажи товара. Режим отложенного применения цен дает возможность подготовить товар к продаже.

Чтобы настроить режим отложенного применения цен, нужно сделать следующее:

· Установить флажок Применение цен на странице Маркетинг раздела Администрирование.

· Установить флажок Использовать применение цен в группе настроек Ценообразование в карточках тех магазинов, где будет применяться данная технология.

Список документов применения цен можно открыть по ссылке Маркетинг – Применение цен - Действующие цены.

Для удобства работы Пользователь может отфильтровать список документов по магазину, ответственному выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

После проведения документа по ссылке Переоценка товаров из меню Печать можно распечатать список переоцененных товаров для конкретного магазина и склада. Также можно вызвать формы для печати ценников и этикеток.

Новый документ Применение цен номенклатуры можно создать на основании документа Установка цен номенклатуры, с помощью команды Применение цен номенклатуры из панели действий раздела Маркетинг, а также из списка документов с помощью кнопки Создать в командной панели списка.

Действия при проведении документа

Заполнение шапки документа

В шапке нового документа нужно заполнить следующие поля:

· дату документа, с которой вступят в действие цены, указанные в документе;

· магазин, в котором производится переоценка номенклатуры.

По умолчанию указывается дата создания документа и значение магазина из настроек пользователя.

Заполнение табличной части документа

Если документ Применение цен номенклатуры формируется на основании документа Установка цен номенклатуры, то табличная часть заполняется из документа-основания. Иначе табличная часть будет заполнена теми позициями номенклатуры, у которых действующие на дату формирования документа цены отличаются от цен, введенных в действие документом Установка цен номенклатуры. При этом в табличную часть выводятся текущие действующие цены, установленные цены номенклатуры и процент изменения цен.

Если для номенклатуры предусмотрено ведение учета по характеристикам, информация выводится в табличную часть с детализацией до характеристик.

С помощью сервиса Заполнить можно автоматически перезаполнить табличную часть номенклатурой, у которой действующие цены отличаются от установленных.

При наличии прав редактирования цен можно перейти в режим редактирования вводимых в действие цен, нажав кнопку Редактировать цены в командной панели табличной части.

Действия при проведении документа

После проведения документа цены номенклатуры переводятся в действующие цены магазина и могут быть переданы на кассы ККМ.
