Ордер на отражение результатов пересчетов товаров

Документ Ордер на отражение результатов пересчетов товаров предназначен для приведения количественных учетных остатков к фактическим при выявлении излишков и недостач в результате инвентаризации.

В розничных сетях с быстрым оборотом товаров для правильного составления заказов поставщику очень важно иметь информацию о фактических остатках товаров на полках. При этом на сотрудников магазина возлагается ответственность за достоверность информации об остатках, а принятие решения о способах оформления выявленных расхождений остается за сотрудниками центрального офиса.
Для реализации этой управленческой схемы в карточке магазина необходимо установить флаг Использовать ордерную схему при отражении излишков и недостач. Сотрудники магазинов для приведения учетных остатков к фактическим оформляют документы складского учета, в частности Ордер на отражение результатов пересчетов товаров, а документы управленческого учета Пересортица товаров, Списание товаров, Оприходование товаров оформляются сотрудниками офиса.

Список ордеров на отражение результатов пересчета товаров можно открыть по ссылке Склад - Инвентаризация - Отражение результатов пересчетов. Пользователь может отфильтровать список ордеров по магазину, складу выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные ордера, задавая поиск по значениям различных реквизитов.

Из меню Печать можно сформировать и распечатать нерегламентированную форму Отражение результатов пересчета, содержащую список товаров с отклонениями.

Документы, создаваемые на основании ордера на отражение результатов пересчета

После проведения ордера на его основании можно создать документы управленческого учета.

Если в ордере были зафиксированы излишки и недостачи по товарам с близкими параметрами, то на его основании можно создать документ Пересортица товаров. Но в этом случае табличную часть создаваемого документа придется заполнять вручную. Для полуавтоматического зачета пересортицы можно воспользоваться сервисом Зачет пересортицы в исходном приказе на пересчет.

Если в ордере были зафиксированы недостачи, то на его основании можно создать документ Списание товаров.

Если в ордере были зафиксированы излишки, то на его основании можно создать документ Оприходование товаров.

Новый документ Ордер на отражение результатов пересчета товаров может быть создан только на основании документа Приказ на пересчет товаров. Перед созданием ордера по данному приказу-основанию должны быть оформлены документы Пересчет товаров. На основании одного приказа может быть создано несколько ордеров.

При создании документа поля Магазин, Склад и Организация заполняются значениями, указанными в приказе-основании. В поле Основание доступна ссылка на приказ-основание.

Заполнение табличной части

Табличная часть документа Ордер на отражение результатов пересчетов товаров при создании ордера автоматически заполняется товарами с выявленными отклонениями из документов Пересчет товаров, которые были оформлены на основании исходного приказа на пересчет. Если по исходному приказу уже были оформлены ордера, то зафиксированные в них отклонения в данный ордер не попадают.

Отрицательное количество в поле Отклонение строки табличной части означает недостачу товара, а положительное - излишек.

Список товаров с отклонениями для данного ордера можно ограничить с помощью отбора товаров. Для этого нужно задать параметры в поле Отбор и выполнить команду Заполнить по отклонениям из командной панели табличной части.

Добавить товары в табличную часть Пользователь может вручную с помощью кнопки Добавить в командной панели табличной части. Кроме того, Пользователь может редактировать поля Номенклатура, Характеристика (если используются характеристики) и Отклонение в строках табличной части.
