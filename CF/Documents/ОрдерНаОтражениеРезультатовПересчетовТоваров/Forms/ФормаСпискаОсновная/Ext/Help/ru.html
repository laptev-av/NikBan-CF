Ордера на отражение результатов пересчетов товаров

Документ Ордер на отражение результатов пересчетов товаров предназначен для приведения количественных учетных остатков к фактическим при выявлении излишков и недостач в результате инвентаризации.

В розничных сетях с быстрым оборотом товаров для правильного составления заказов поставщику очень важно иметь информацию о фактических остатках товаров на полках. При этом на сотрудников магазина возлагается ответственность за достоверность информации об остатках, а принятие решения о способах оформления выявленных расхождений остается за сотрудниками центрального офиса.
Для реализации этой управленческой схемы в карточке магазина необходимо установить флаг Использовать ордерную схему при отражении излишков и недостач. Сотрудники магазинов для приведения учетных остатков к фактическим оформляют документы складского учета, в частности Ордер на отражение результатов пересчетов товаров, а документы управленческого учета Пересортица товаров, Списание товаров, Оприходование товаров оформляются сотрудниками офиса.

Документ Ордер на отражение результатов пересчета товаров может быть создан только на основании документа Приказ на пересчет товаров. Перед созданием ордера по данному приказу-основанию должны быть оформлены документы Пересчет товаров. На основании одного приказа может быть создано несколько ордеров.

Список ордеров на отражение результатов пересчета товаров можно открыть по ссылке Склад - Инвентаризация - Отражение результатов пересчетов.

Для удобства работы Пользователь может установить быстрый отбор по полям Магазин и Склад в шапке списка документов. Кроме того, с помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов, например, по приказу-основанию.
