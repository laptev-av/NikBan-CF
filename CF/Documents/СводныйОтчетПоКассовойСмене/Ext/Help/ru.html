Сводные отчеты по кассовой смене

Документ предназначен для формирования сводного отчета по магазину. Документ формируется на основе документов Отчет о розничных продажах, ранее сформированных при закрытии кассовой смены по каждой кассе ККМ магазина.

Список документов можно открыть по ссылке Продажи – Розничные продажи - Сводные отчеты по кассовой смене. Для удобства работы Пользователь может отфильтровать список отчетов по кассе выбрав нужные значения. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов. Выбранный документ может быть напечатан по форме КМ-7 - Сведения о показаниях счетчиков контрольно-кассовых машин и выручке организации.

Новый документ можно добавить с помощью кнопки Создать в командной панели списка. Табличная часть документа заполняется путем подбора документов Отчет о розничных продажах.
