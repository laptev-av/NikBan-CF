Ордер на отражение недостач товаров

Документ Ордер на отражение недостач товаров предназначен для приведения количественных учетных остатков к фактическим при выявлении недостач товаров на складе без проведения инвентаризации.

Для реализации ордерной схемы отражения недостач в карточке магазина необходимо установить флаг Использовать ордерную схему при отражении излишков и недостач. В этом случае сотрудники магазинов для приведения учетных остатков к фактическим оформляют документы складского учета, в частности Ордер на отражение недостач товаров, а документы управленческого учета Списание товаров оформляются сотрудниками офиса.

Список ордеров на отражение недостач и излишков товаров можно открыть по ссылке Склад - Складские операции - Излишки, недостачи. Для удобства работы Пользователь может отфильтровать список по магазину, складу выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

 

На основании выбранного документа можно создать документ управленческого учета Списание товаров.

Из меню Печать можно сформировать и распечатать нерегламентированную форму Отражение результатов недостач, содержащую список товаров с количеством к списанию.

Новый документ Ордер на отражение недостач товаров можно создать с помощью команды Отражение недостач из меню Создать командной панели списка документов.

Заполнение шапки документа

При создании ордера некоторые поля шапки документа могут быть автоматически заполнены значениями, установленными для пользователя по умолчанию, или теми значениями, которые были выбраны пользователем для быстрого отбора в списке документов. Пользователь редактирует или заполняет следующие поля:

· Магазин, в котором обнаружена недостача;

· Склад выбранного магазина, на котором обнаружена недостача;

· Организация, по товарам которой обнаружена недостача.

Заполнение табличной части

Добавить товары в табличную часть можно следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· загрузить список товаров из терминала сбора данных, в этом случае, если табличную часть предварительно не очистить, то сведения по количеству товара из ТСД просуммируются с количеством в соответствующих товарных строках табличной части;

· по штрихкоду. Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне. Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1.

Ввод серий товаров

Серию товара необходимо вводить в строке табличной части документа Ордер на отражение недостач товаров при выполнении нескольких условий:на предприятии ведется учет товара по сериям, для данного вида номенклатуры в данном магазине настроена политика учета серий, согласно этой политике нужно учитывать серии при отражении недостач.
В этом случае Пользователь должен выделить строку, для которой он будет вводить серии, и с помощью кнопки Указать серии в командной панели табличной части открыть форму Ввод серий товаров и заполнить её. После корректного ввода серии она отображается в выделенной строке в поле Серия.
