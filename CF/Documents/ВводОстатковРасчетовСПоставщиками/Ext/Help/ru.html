Ввод остатков расчетов с поставщиками

Документ "Ввод остатков расчетов с поставщиками" предназначен для ввода первоначальной суммы задолженности в разрезе магазинов. Если не представляется возможным установить суммы задолженности в разрезе магазинов (документов поступления), то можно отнести задолженность на один определенный магазин, из кассы которого производить погашение.

Список документов открывается Администрирование - Первоначальное заполнение ИБ - Ввод остатков расчетов с поставщиками.

Документ Ввод остатков расчетов с поставщиками создается с помощью нажатия кнопки Создать.

В шапке указывается магазин, в табличную часть документа с помощью кнопки Добавить вводится информация о Поставщике, документе расчета и суммах.
