Подбор по остаткам товаров у отправителя

Список заполняется теми остатками, которые числятся на организации-отправителе и складе, выбранных в шапке документа Передача товаров, на текущую дату.

Пользователь может изменить дату, тогда список автоматически заполнится остатками на выбранную дату.

Для убыстрения подбора можно использовать кнопки в командной панели списка, с помощью которых можно выделить все строки, исключить все строки из выбора, выбрать выделенные строки и исключить выделенные строки из выбора.

В выбранных строках можно изменить количество товара для передачи. По умолчанию это количество равно остатку.

Отобранные товары переносятся в табличную часть документа Передача товаров с помощью кнопки Перенести в документ.

Подбор по отрицательным остаткам у получателя

Список формируется по отрицательным остаткам, образовавшимся у организации-получателя в результате продаж товаров, которые числились на организации-отправителе.

Работа со списком выполняется аналогично работе со списком остатков отправителя.
