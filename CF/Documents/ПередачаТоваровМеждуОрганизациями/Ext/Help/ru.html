Передача товаров между организациями

Документ Передача товаров используется для того, чтобы переоформить товар на другую организацию, не меняя при этом его физического местоположения. Товар передается между организациями на условиях купли- Form.Продажи. С помощью документа можно оформить передачу до реализации. Необходимость в такой операции может возникнуть, например, при перемещении товара между магазинами, принадлежащими разным организациям. Если товар был продан от имени одной организации, а числился на другой, то можно оформить передачу после реализации.

Список документов можно открыть по ссылке Закупки - Снабжение - Передачи товаров. Пользователь может отфильтровать список документов по магазину, складу, отправителю, получателю выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Выбранный документ может быть напечатан по форме:

· Накладная на отпуск товаров на сторону по форме М-15

· Товарная накладная по форме ТОРГ-12

· Счет-фактура

· УПД (Универсальный Передаточный Документ)

 

Новый документ можно добавить с помощью кнопки Создать в командной панели списка или может быть сформирован на основании документа Перемещение товаров.

Заполнение шапки документа

Если документ формируется на основании документа Перемещение товаров, то поля Отправитель, Магазин и Склад заполняются значениями полей Организация, Магазин получатель и Склад получатель из документа-основания. Товары и их количество тоже переносятся из документа-основания.

При непосредственном создании нового документа нужно выбрать организации, между которыми передается товар, в полях Отправитель и Получатель, а также место хранения товара в полях Магазин и Склад. По умолчанию эти поля заполняются в соответствии с настройками Пользователя, а если они не установлены, то значениями полей отбора из списка документов. Если справочник содержит единственный элемент, то он будет подставлен по умолчанию.

Кроме того, пользователь должен выбрать тип операции (Передача до или после реализации) и может выбрать вид цены при передаче.

Виды цен при передаче

После указания организации-отправителя на закладке Дополнительно заполняется информация о применяемом этой организацией способе налогообложения, в том числе значение флага Цена включает НДС.
В списке Вид Цены отображаются только те виды цен, у которых установлен флаг Использовать при передаче между организациями (Маркетинг-Ценообразование-Виды цен) при условии, что у этих видов цен значение флага Цена включает НДС совпадает со значением этого флага на закладке Дополнительно в документе.

Добавление товаров в табличную часть

Товары в табличную часть документа можно добавить следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· с помощью сервиса Подобрать по остаткам товаров у отправителя из меню Заполнить в командной панели табличной части. Список заполняется теми остатками, которые числятся на организации-отправителе и складе, выбранных в шапке документа Передача товаров, на текущую дату. Пользователь может изменить дату, тогда список автоматически заполнится остатками на выбранную дату. Для убыстрения подбора можно использовать кнопки в командной панели списка, с помощью которых можно выделить все строки, исключить все строки из выбора, выбрать выделенные строки и исключить выделенные строки из выбора. В выбранных строках можно изменить количество товара для передачи. По умолчанию это количество равно остатку. Отобранные товары переносятся в табличную часть документа Передача товаров с помощью кнопки Перенести в документ.

· по штрихкоду. Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне. Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1.

Заполнение цен

Цены в документе можно заполнить с помощью сервиса Заполнить цены по виду цен из списка Заполнить в командной панели табличной части.
