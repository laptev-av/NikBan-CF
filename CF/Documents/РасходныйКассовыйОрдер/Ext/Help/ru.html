Расходный кассовый ордер

Документ Расходный кассовый ордер (РКО) предназначен для учета расхода наличных денежных средств.

С помощью РКO можно зарегистрировать выдачу денежных средств (ДС) из кассы организации по следующим операциям:оплата поставщику, сдача ДС в банк, выдача ДС в другую кассу, возврат оплаты покупателю, выдача ДС в кассу ККМ, прочие расходы, выдача ДС в другую организацию, выплата заработной платы сотруднику, выплата заработной платы по ведомости.

Выплаты регистрируются в разрезе предустановленных хозяйственных операций, аналитика которых может быть расширена с помощью Статей движения денежных средств.

Список документов РКО открывается Финансы – Расходные кассовые ордера. Для удобства работы Пользователь может установить отбор по полям Организация, Касса, Операция. По каждому документу в списке дополнительно выводится информация о Дате, Операция, Сумма, документа. На закладке Выплаты зарплаты можно просмотреть список выплат, который формируется по открытым ведомостям и выводится на экран кассиру (см. Выплата заработной платы по ведомости). С помощью команды Печать для каждого выбранного документа может быть распечатан:

· Расходный кассовый ордер;

· Квитанция к сумке;

· Накладная к сумке;

· Препроводительная ведомость к сумке;

· Препроводительная ведомость и накладная к сумке.

Документ РКО оформляется по ссылке Финансы – Денежные средства - Расходные кассовые ордера – кнопка Создать.

Оплата поставщику

Выплата поставщику может производиться на основании документов Поступление от поставщика или Заказ поставщику. При этом реквизиты документа, за исключением кассы, из которой планируется выдача денежных средств, заполнятся по документу-основанию.

В табличной части документа Расходный кассовый ордер можно выбрать несколько документов указанного поставщика, по которым нужно произвести выплаты. При этом в табличной части отобразится сумма документа и сумма НДС по документу. Пользователь может отредактировать суммы выплат по документам, вводя нужные значения в поле Сумма. При этом общая сумма расходного кассового ордера будет пересчитана автоматически.

Кассир операционной кассы может оформить выплаты поставщикам на основании сформированных менеджером этапов оплат в документах Заказ поставщику и Поступление товаров.

Для этого в списке Расходные кассовые ордера (Финансы – Денежные средства) необходимо создать новый документ и указать поставщика, которому предназначается выплата. При нажатии кнопки Подобрать неоплаченные открывается список документов данного поставщика, имеющих неоплаченные на момент создания РКО запланированные этапы оплат. По умолчанию выводятся наличные оплаты, но кассир может изменить форму оплаты или вывести список вне зависимости от запланированной формы оплаты.

Для каждого документа выводится сумма документа и сумма этапа оплаты. Кассир может ввести выделенную для оплаты сумму и с помощью сервиса Распределить сумму платежа рассчитать суммы, причитающиеся на каждый документ в списке. Другой вариант – отметить нужные документы и перенести в табличную часть РКО с помощью команды Перенести в документ. При нажатии кнопки Сохранить общая сумма расходного кассового ордера пересчитается автоматически.

Сдача ДС в банк

Нужно выбрать кассу, счет организации, на который будет зачислена инкассируемая сумма, и номер сумки инкассатора, который закреплен за магазином. Сумма сдаваемых денежных средств будет рассчитана по заполненной табличной части описи сдаваемых купюр.

При оформлении поступления денежных средств из банка, например, разменной монеты или средств на выплату заработной платы, пользователю нужно указать расчетный счет организации, с которой выбыла указанная сумма, и кассу, на которую приходуются денежные средства. Печатные формы "Препроводительная ведомость к сумке" и "Накладная к сумке" будут содержать QR-код, с информацией о данных печатной формы об инкассации денежных средств.

Выдача ДС в другую кассу

Осуществляется только между кассами одной организации, что контролируется при заполнении полей документа. Передача денежных средств между кассами предприятия производится в два этапа:

в кассе-отправителе оформляется Расходный кассовый ордер на нужную сумму;

после фактического поступления денег в кассе получателе оформляется Приходный кассовый ордер на основании Расходного документа.

Передача денежных средств между кассами одной организации производится по операции Выдача ДС в другую кассу. При этом Организация документа определяется по кассе-отправителю, а в качестве кассы-получателя можно выбрать Кассы только этой организации.

Внутренняя передача ДС

Предназначенная для передачи ДС между касс головной организации и обособленного подразделения или между кассами разных обособленных подразделений.

Флаг "обособленное подразделение" устанавливается в справочнике Организации.

Возврат оплаты покупателю

В этом случае реквизиты заполняются вручную, а документ-основание указывать не обязательно. В качестве покупателя может быть указан как контрагент, так и розничный покупатель.

По оформленному и проведенному документу может быть напечатан чек на подключенном к данному рабочему месту оборудовании - фискальном принтере или принтере документов. Для этого нужно нажать кнопку Напечатать чек в командном меню документа.

Выдача ДС в кассу ККМ

Используется для оформления выдачи разменной монеты в кассы ККМ. При этом организация определяется по Кассе-отправителю и в качестве получателя размена можно указать только Кассы ККМ определенной организации. По проведенному документу РКО на рабочем месте кассира (РМК) производится операция Внесение.

Прочие расходы

Эта операция предусмотрена для оформления расходов денежных средств по другим нуждам. При этом предприятие имеет возможность анализировать прочие расходы с нужной для руководства предприятием детализацией, используя справочник Статьи движения денежных средств.

Выдача ДС в другую организацию

Используется для передачи ДС между организациями. Для данной операции никаких ограничений на выбор кассы-получателя не накладывается.

Выплата заработной платы сотруднику

Выплата осуществляется по платежной ведомости, если она не существует ее можно создать. После выбора платежной ведомости, необходимо выбрать сотрудника и заполнить другие поля. После проведения РКО в соответствующей строке ведомости появится запись о документе, по которому была произведена данная выплата.

Выплата заработной платы по ведомости

Ведомость на выплату зарплаты формируется в центральном офисе торговой сети в конфигурации «Бухгалтерия предприятия» и передается в конфигурацию «Розница» в процессе обмена данными. В случае, когда для автоматизации розничного предприятия используется только «Розница», ведомость на выплату зарплаты создается вручную с помощью кнопки Создать в списке Ведомости на выплату зарплаты.

Подготовленную ведомость можно закрыть, создав на основании ведомости Расходный кассовый ордер. Реквизиты РКО заполнятся по данным ведомости, сумма ордера рассчитается только по выплатам, произведенным по ведомости. Ведомости, по которым сформирован расходный кассовый ордер, нельзя редактировать.

Возврат аванса с помощью РКО

Можно также оформить возврат аванса с помощью Расходного кассового ордера или Эквайринговой операции с операцией Возврат оплаты покупателю.
