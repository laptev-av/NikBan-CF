Отгрузка - Расходный ордер

Документ Расходный ордер на товары предназначен для фиксирования фактического количества отгружаемого со склада товара при использовании ордерной схемы отгрузки товаров.

Список расходных ордеров можно открыть по ссылке Склад – Складские операции - Отгрузка. В этой же рабочей области доступна страница Распоряжения на отгрузку, где представлен список документов, которые являются основаниями для отгрузки товаров. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные ордера, задавая поиск по значениям различных реквизитов. Выбранный ордер может быть напечатан как расходный ордер. С помощью команды Печать можно напечатать Расходный ордер на товары. Список товаров без указания цен и сумм можно также просмотреть в печатной форме Опись номенклатуры.

Документ Расходный ордер на товары можно оформить на основании документов: Возврат товаров поставщику, Перемещение товаров, Реализация товаров. Для создания ордера можно воспользоваться списком документов-оснований на странице Распоряжения на отгрузку: выделить нужный документ и в командной панели списка нажать кнопку Создать расходный ордер. Кроме того, можно создать ордер с помощью команды Расходный ордер из меню Создать на основании конкретного документа-основания или списка соответствующих документов.

Заполнение документа Расходный ордер на товары

При создании расходного ордера значения полей шапки документа Магазин и Склад, а также товарные строки табличной части будут заполнены из документа-основания.

Проверка фактического количества отгружаемого товара

Внести в документ фактическое количество товара можно в форме, которая открывается при нажатии кнопки Проверка количества товара командной панели табличной части документа.

Добавление товаров в табличную часть

Товары, которые фактически отгружаются, но отсутствуют в документе-основании, можно добавить в расходный ордер следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· с помощью сервиса Подобрать товары;

· по штрихкоду. Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне. Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1.

Редактирование данных в строке табличной части

В каждой товарной строке пользователь может ввести или изменить данные в следующих полях:

· Характеристика номенклатуры - можно выбрать из списка характеристик данной номенклатуры;

· Упаковка - по умолчанию указывается основная единица измерения, но может быть выбрана любая упаковка данной номенклатуры;

· Количество - количество поступивших единиц и упаковок.

Ввод серии товара

Серию товара необходимо вводить в строке расходного ордера при выполнении нескольких условий:на предприятии ведется учет товара по сериям, для данного вида номенклатуры и для данного магазина настроена политика учета серий, согласно этой политике по операции, в рамках которой создан данный ордер, нужно вводить серии. В этом случае в строке товара в поле Количество будет отображен значок Статусы серии.
Значок красного цвета означает, что серии еще не введены. Для ввода серии нужно из поля Количество открыть форму Ввод количества и серий товаров и заполнить её.
После корректного ввода серии и количества, значок серии меняет свой цвет на зеленый.

Ввод серийного номера подарочного сертификата

Если в строку табличной части расходного ордера внесен подарочный сертификат, то в поле Количество отображается значок подарочного сертификата. Для ввода серийного номера сертификата из поля Количество нужно открыть форму Ввод подарочных сертификатов. В открывшейся форме можно указать диапазон серийных номеров. Диапазон можно выбрать из существующих или создать новый.

Передача данных в ГИСМ о других операциях с товаром

Все расходные складские операции с маркированным товаром, такие как списание, возврат поставщику или от покупателя, отгрузка оптом сопровождаются отправкой соответствующего Уведомления в систему ГИСМ. Ожидаемые поступления товаров от поставщика сопровождаются уведомлениями о поступлении из ГИСМ, которое организация-покупатель должна подтвердить. Для этого после проведения документа движения маркированного товара появляется ссылка, с помощью которой пользователь может отправить уведомление об операции в ГИСМ. Рассмотрим процесс взаимодействия с ГИСМ на примере оптовой отгрузки и поступления товаров.
