Расчеты и установки себестоимости

Документ Установка себестоимости предназначен для ручной корректировки информации о себестоимости товаров. Данную возможность можно использовать в том случае, если себестоимость товаров можно определить только экспертным путем, например, при переходе на конфигурацию Розница неавтоматизированных магазинов для ввода начальной информации о себестоимости товаров.

Список документов открывается Закупки – См. также -- Расчеты и установки себестоимости.

Новый документ вводится с помощью кнопки Создать. В документ вводится информация о товарах и себестоимости единицы товара (Цена). Документ фиксирует одинаковую себестоимость товаров для всех магазинов, зарегистрированных в информационной базе конфигурации Розница.

При оформлении документа предусмотрены возможности по заполнению и изменению списка товаров в документе по штрих коду и через Подбор товаров.

Поиск по штрихкоду- в появившемся диалоговом окне можно задать штрихкод товара. Программа найдет товар по штрихкоду и добавит товар в табличную часть документа. Предусмотрено также заполнение табличной части документа с помощью сканера штрихкода.
