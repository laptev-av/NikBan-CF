Акт о расхождениях при приемке товаров

Для фиксации расхождений при приемке служит Акт о расхождениях при приемке товаров. Для фиксации расхождений необходимо из списка документов Перемещение товаров (Закупки – Снабжение) выбрать нужный и на его основании сформировать документ Акт о расхождении при приемке товаров.

В табличной части сформированного акта необходимо исправить фактическое количество недостающих или избыточно прибывших товаров и указать, где именно следует учесть расхождения:на складе магазина-отправителя, либо на складе магазина-получателя. Если транспортировку производит внешний перевозчик, есть возможность учесть недостачи, возникшие за счет компании-перевозчика. Принимать решение о месте учета расхождений непосредственно при формировании Акта не обязательно - зафиксировать принятое решение можно позже, после проведения пересчета на складе-отправителе. Документы и товарные позиции, по которым еще не принято решение можно просмотреть с помощью отчета Неустановленные расхождения (Закупки - Отчеты по запасам и закупкам - Снабжение).

Список актов открывается по ссылке Закупки – Снабжение – Акты о расхождениях. Для удобства работы Пользователь может отфильтровать список выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов. Выбранный документ может быть напечатан по форме Акт расхождений по перемещению.

На основании выбранного акта можно создать документы Ордер на отражение излишков, Ордер на отражение недостач, Оприходование товаров, Списание товаров.
