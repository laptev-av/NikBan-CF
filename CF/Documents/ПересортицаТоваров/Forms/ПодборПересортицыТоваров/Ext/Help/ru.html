Подбор пересортицы товаров

Для корректной работы помощника по зачету пересортицы (открывается по ссылке Склад – Инвентаризация - Приказы на перерасчет товаров – Сервис – Зачет пересортицы) пользователь должен сначала оформить результаты пересчета товаров на основании исходного приказа: провести документы Пересчет товаров и, в случае проведения инвентаризации по ордерной схеме, провести ордера на отражение результатов пересчета. Если при пересчете зафиксированы излишки и недостачи, то для каждого товара с излишком помощник подбирает для взаимозачета товары с недостачей в порядке убывания приоритета. Приоритет параметров для зачета пересортицы настраивает администратор в форме Параметры анализа пересортицы на странице Запасы и Закупки раздела Администрирование.

Зачет недостачи производится последовательно для каждой позиции излишка. Товар с излишком отображается в шапке формы помощника, товары с недостачей - в табличной части, по убыванию приоритета, показывающего, насколько хорошо данная позиция недостачи соответствует позиции излишка. Пользователь может выбрать позицию или несколько позиций с недостачами для зачета излишка. Количество единиц приходуемого и списываемого товара должно совпадать. Пользователь переходит к следующему товару с излишком с помощью кнопки Далее в нижней части формы. Позиции недостачи, зачтенные на предыдущем шаге, становятся недоступными для выбора. Таким образом, пользователь отбирает на каждом шаге наиболее подходящие для взаимозачета позиции.

Процесс может быть прекращен на любом шаге с помощью кнопки Завершить. В результате будет создан документ Пересортица товаров с табличной частью, заполненной товарами, которые пользователь выбрал для взаимозачета.
