Пересортица товаров

Документ Пересортица товаров предназначен для одновременного списания и оприходования равного количества товаров с близкими параметрами. В соответствии с положением о бухгалтерском учете можно взаимно зачесть равное количество излишков и недостач для товаров одного и того же наименования, но разного вида или сорта.

Список документов открывается по ссылке Склад – Инвентаризация – Пересортица товаров. Для удобства работы Пользователь может отфильтровать список документов по магазину, складу. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов. Выбранный документ может быть напечатан по форме Акт о пересортице товаров.

Новый документ можно добавить с помощью кнопки Создать в командной панели списка. Если инвентаризация в магазине проводится по ордерной схеме, то пересортицу можно оформить на основании документа Ордер на отражение результатов пересчетов товаров (из меню Создать на основании, выбрать пункт Пересортица товаров. При этом табличную часть документа пользователь заполняет вручную). Кроме того, документ Пересортица товаров можно создать на основании документа Приказ на пересчет товаров (по ссылке Зачет пересортицы из меню Сервис. При этом запустится помощник создания документа Пересортица товаров.).

Заполнение шапки документа

Если документ был создан на основании приказа на пересчет товаров или ордера на отражение результатов пересчета, то поля шапки заполняются автоматически значениями полей документа-основания.

При создании документа Пересортица товаров вручную некоторые поля шапки документа могут быть автоматически заполнены значениями, установленными для пользователя по умолчанию, или теми значениями, которые были выбраны пользователем для быстрого отбора в списке документов. Пользователь редактирует или заполняет следующие поля:

· Магазин, в котором производится зачет пересортицы;

· Склад выбранного магазина, на котором производится зачет пересортицы;

· Организация, по товарам которой производится зачет пересортицы.

Заполнение табличной части

Если документ Пересортица товаров был создан с использованием помощника зачета пересортицы, то табличная часть заполняется товарами, выбранными пользователем для взаимозачета в процессе работы с помощником. Иначе пользователь заполняет табличную часть, вручную подбирая и списываемый, и приходуемый товар, и количество взаимозачета.

Пользователь может указать цены для зачета пересортицы вручную или с помощью следующих сервисов, расположенных в меню Цены командной панели табличной части:

· заполнить цены по розничным ценам,

· заполнить цены по себестоимости,

· заполнить цены по виду цен.
