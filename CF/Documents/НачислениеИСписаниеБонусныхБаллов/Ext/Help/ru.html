Начисление и списание бонусных баллов

Документ Начисление и списание бонусных баллов предназначен для начисления или списания бонусных баллов, не связанных с продажами товаров.

Документ может создаваться как вручную, так и автоматически с помощью регламентного задания по установленным правилам расчета бонусных баллов.

Список документов можно открыть по ссылке Маркетинг - Бонусные программы - Начисления и списания баллов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов. На основании выбранного документа можно создать E-mail или SMS рассылки.

Новый документ можно ввести с помощью кнопки Создать командной панели списка документов.

Каждое начисление или списание соответствует одной бонусной программе лояльности, которая выбирается в соответствующем поле. Правило начисления из числа относящихся к выбранной бонусной программе может быть использовано для автоматического заполнения параметров периода использования баллов и таблицы начисления или списания бонусных баллов. Для этого предусмотрена команда Заполнить по правилу в командной панели табличной части документа. Заполненные по правилу параметры могут быть отредактированы пользователем.
