Приходные ордера на товары

Документ Приходный ордер на товары предназначен для фиксирования фактического количества принимаемого товара при оформлении поступления товаров по ордерной схеме.

Список документов открывается Склад - Складские операции – Приемка. Для удобства работы Пользователь может отфильтровать список документов по магазину, складу выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов.

Для печати приходного ордера нужно нажать кнопку Печать в командной панели выбранного документа. Список товаров без указания цен и сумм можно также просмотреть в печатной форме Опись номенклатуры.

Новый документ можно добавить с помощью кнопки Создать приходный ордер расположенной на закладке Распоряжения на приемку, где представлен список документов, которые являются основаниями для приемки товаров:Возврат товаров от покупателя, Заказ поставщику, Перемещение товаров, Поступление товаров.

Для создания ордера нужно выделить документ и в командной панели списка нажать кнопку Выполнить приемку. Кроме того, можно создать ордер с помощью команды Приходный ордер из меню Создать на основании конкретного документа-основания или списка соответствующих документов.

Заполнение документа Приходный ордер на товары

При создании приходного ордера значения полей шапки документа Магазин и Склад, а также товарные строки табличной части будут заполнены из документа-основания.

Внести в документ фактическое количество товара можно в форме, которая открывается при нажатии кнопки Проверка количества товара командной панели табличной части документа.

Добавление товаров в табличную часть

Товары, которые фактически поступили, но отсутствуют в документе-основании, можно добавить в приходный ордер следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· с помощью сервиса Подобрать товары;

· по штрихкоду. Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне. Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1

· по магнитному коду

Редактирование данных в строке табличной части

В каждой товарной строке пользователь может ввести или изменить данные в следующих полях:

· Характеристика номенклатуры - можно выбрать из списка характеристик данной номенклатуры;

· Упаковка - по умолчанию указывается основная единица измерения, но может быть выбрана любая упаковка данной номенклатуры;

· Количество - количество поступивших единиц и упаковок.

Ввод серии товара

Серию товара необходимо вводить в строке приходного ордера при выполнении нескольких условий:на предприятии ведется учет товара по сериям, для данного вида номенклатуры и для данного магазина настроена политика учета серий, согласно этой политике по операции, в рамках который создан данный ордер, нужно вводить серии. В этом случае в строке товара в поле Количество будет отображен значок Статусы серии. Значок красного цвета означает, что серии еще не введены. Для ввода серии нужно из поля Количество открыть форму Ввод количества и серий товаров и заполнить её. После корректного ввода серии и количества, значок серии меняет свой цвет на зеленый.

Ввод серийного номера подарочного сертификата

Если в строку табличной части приходного ордера внесен подарочный сертификат, то в поле Количество отображается значок подарочного сертификата.
Для ввода серийного номера сертификата из поля Количество нужно открыть форму Ввод подарочных сертификатов. В открывшейся форме можно указать диапазон серийных номеров. Диапазон можно выбрать из существующих или создать новый.
