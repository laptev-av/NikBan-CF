Цены номенклатуры (установка цен номенклатуры)

Документ Установка цен номенклатуры служит для расчета и назначения постоянных цен номенклатуры по различным видам цен.

Список документов установки цен можно открыть по ссылке Маркетинг - Ценообразование - Цены номенклатуры.

С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Выбранный документ может быть напечатан. Из меню Печать можно выполнить следующие действия:

· По ссылке Переоценка товаров можно распечатать список переоцененных товаров с остатками по конкретному магазину и складу.

· По ссылке Установка цен номенклатуры можно распечатать список всех переоцененных товаров по одному или по нескольким видам цен, которые пользователь может выбрать из установленных в данном документе.

· Также можно вызвать формы для печати ценников и этикеток.

Новый документ Установка цен номенклатуры можно создать на основании документов Изменение ассортимента и Поступление товаров, с помощью команды Установка цен номенклатуры из панели действий раздела Маркетинг, а также из списка документов с помощью кнопки Создать в командной панели списка.

При создании документа Установка цен номенклатуры в шапке документа указывается дата начала действия цен. Ввод документа с датой большей, чем дата предыдущего документа, отменяет действие цен предыдущего документа и вводит в действие новые цены.

Выбор видов цен для расчета

Если документ Установка цен номенклатуры создается вручную, то сначала открывается табличная часть, в которой можно выбрать виды цен из тех, что зарегистрированы в справочнике Виды цен и доступны пользователю. В одном документе можно установить цены по нескольким видам. Необходимо отметить те виды цен, по которым будут изменены цены.
Если документ Установка цен номенклатуры создан на основании документов Изменение ассортимента или Поступление товаров, то открывается табличная часть с номенклатурой, загруженной из документов-оснований, и с соответствующим составом видов цен для установки. Этот состав можно изменить, перейдя на страницу видов цен с помощью кнопки Перейти к изменению состава видов цен внизу табличной части.

Чтобы отметить все цены или снять установленные отметки, можно воспользоваться сервисными кнопками командной панели.

Те виды цен, которые влияют на расчет (т. е. являются ценой-основанием для расчета) выбранных пользователем, будут отмечены в колонке Влияющие виды цен.
Те виды цен, которые зависят от выбранных (т. е. рассчитываются на основании выбранных), будут отмечены в колонке Зависимые виды цен.
С помощью кнопок командной панели Выбрать все влияющие и Выбрать все зависимые можно отобрать все связанные цены, чтобы пересчитать их значения в одном документе.

Если цена рассчитывается по данным информационной базы с помощью схемы компоновки, то параметры расчета можно изменить непосредственно в документе Установка цен номенклатуры. Для этого можно воспользоваться кнопкой командной панели Редактировать настройки. Восстановить настройки можно с помощью кнопки Восстановить настройки по умолчанию.

Для установки цен номенклатуры по выбранным видам необходимо перейти в следующее окно нажатием кнопки Перейти к установке цен в подвале страницы.

Заполнение списка номенклатуры для назначения цен

· Заполнение вручную. Для того чтобы ввести новую позицию вручную, необходимо воспользоваться клавишей Insert или кнопкой Добавить в командной панели табличной части. Можно добавить в список номенклатурную позицию с помощью пункта Добавить номенклатуру или характеристику товара с помощью пункта Добавить характеристику, если учет товара ведется по характеристикам.

· Добавить товары можно с помощью сканера штрихкодов, считывателя магнитных кодов или загрузки из терминала сбора данных (ТГСД).

· Сервис Подбор товаров. Добавить товар можно с помощью сервиса Подбор товаров, который вызывается по ссылке Открыть подбор из меню Подобрать товары в командной панели табличной части.

· Заполнение по ценовой группе. С помощью пункта Добавить товары по ценовой группе меню Подобрать товары можно внести в табличную часть все товары одной или нескольких выбранных ценовых групп. Для этого необходимо в открывшемся списке доступных ценовых групп выделить нужные и нажать кнопку Выбрать. При этом номенклатура, по которой ведется учет по характеристикам, будет автоматически внесена в список с детализацией до характеристик.

· Заполнение по отбору. Чтобы отобрать товар по произвольным параметрам, можно воспользоваться пунктом Добавить товары по отбору меню Подобрать товары. В открывшемся окне установить параметры отбора и перенести информацию об этих товарах в табличную часть документа (с установкой фильтров по реквизитам номенклатуры и характеристикам.) Есть возможность отбора по остаткам. Возможно также заполнения табличной части товарами, на которые цены изменились более установленного правилами порога изменения цен. Для этого нужно выбрать пункт Добавить товары по отбору с отклонением наценки выпадающего меню Подобрать товары и затем обычным способом установить нужные параметры отбора.

Ввод информации о ценах

Пользователь может заполнить цены в табличной части документа значениями из информационной базы. Для этого необходимо нажать кнопку Установить цены и выбрать пункт Загрузить текущие назначенные цены.
В открывшейся форме Выбор параметров расчета цен устанавливаются следующие параметры загрузки:

· В табличной части нужно отметить вид загружаемых цен.

· В подвале формы нужно выбрать дату, на которую берется срез цен.

· Можно установить процент изменения цен. Загружаемые цены будут изменены на заданный пользователем процент непосредственно в процессе загрузки.

· Если автоматическое округление цен нежелательно, можно снять флажок Применять округление.

· Можно задать режимы выборочного заполнения документа, позволяющие пользователю работать только с избранными проблемными позициями. Для этого нужно изменить положения переключателей Пересчитать строки документа (Все/Только выделенные) и Пересчитать ячейки документа (Все/Только незаполненные).

Для того, чтобы загрузить цены в документ согласно выбранным параметрам, нужно нажать кнопку Загрузить.

Для видов цен с правилом калькуляции Задавать вручную можно ввести значения цен непосредственно в поле Новая цена табличной части документа.
Введенные вручную цены могут быть округлены в соответствии с правилами округления для данного вида цен. Для этого в выпадающем меню Установить цены нужно выбрать пункт Округлить ручные цены. В открывшейся форме можно отметить нужный вид цен, выбрать режим округления – для всех строк документа или только для предварительно выделенных пользователем и нажать кнопку Округлить.

Расчет значений зависимых видов цен может быть произведен по мере ручного ввода или загрузки цен-оснований. Для этого должен быть установлен флажок Рассчитывать цены автоматически в выпадающем меню Параметры.
Помимо этого, цены могут быть рассчитаны в любой момент с помощью сервиса Рассчитать цены меню Установить цены. При этом открывается форма для расчета цен, в которой можно отметить нужный вид цен, выбрать режим расчета - для всех строк (ячеек) документа или только для выделенных (незаполненных) и нажать кнопку Рассчитать.

Если значение цены-основания не задано, то расчет зависимых от нее цен будет производиться исходя из нулевого значения цены-основания.

Кроме того, можно воспользоваться сервисом Изменить цены на процент меню Установить цены. При этом открывается форма для расчета цен, в которой можно отметить нужный вид цен и процент изменения, выбрать нужно ли применять округление, задать базовую цену расчета, выбрать режим расчета - для всех строк документа или только для выделенных и нажать кнопку Рассчитать.

Особенности заполнения цен.

Если для товаров ведется учет по характеристикам, то цены должны быть зарегистрированы для каждой характеристики позиции номенклатуры. Список товаров с характеристиками удобно представлять в табличной части документа Установка цен в виде дерева или иерархического списка. Представление списка управляется кнопкой Отображение таблицы.
Если цена товара не зависит от значения характеристики, то ее можно ввести в поле Цена группировочной строки без указания характеристики. Эта цена будет автоматически зарегистрирована для всех характеристик товаров. Режим автоматического заполнения цен характеристик в документе устанавливается флагом Заполнять цены характеристик по ценам номенклатуры в меню Параметры.
В дальнейшем можно развернуть список товаров с характеристиками и указать другую цену для характеристики товара, если она отличается от основной цены товара.

Пользователь может назначить цену на упаковку товара, отличную от базовой, выбрав нужное значение поля Упаковка в табличной части документа. При автоматическом расчете цена за упаковку рассчитывается с учетом коэффициента, заданного для данной упаковки.

Колонки с видами цен, недоступными пользователю, но влияющими на расчет доступных цен не отображаются на формах документа Установка цен номенклатуры.

Если установлен флаг Отображать влияющие цены, то у каждой цены, рассчитываемой по другим видам цен, отображается колонка % наценки, в которой выводится процент наценки на влияющую цену. Если влияющих видов цен несколько, то процент наценки не рассчитывается и не отображается.

Если в документе установлена настройка Рассчитывать цены автоматически в выпадающем меню Параметры, то расчет цен при заполнении на основании документов Поступление товаров, Перемещение товаров и Оприходование товаров товародвижения будет осуществляется автоматически.

Регистрация цен

Для регистрации цен на товары и услуги необходимо провести документ Установка цен номенклатуры. Фактически при проведении документа назначенные в документе цены вводятся в действие и могут быть переданы в магазины торговой сети.

Если в силу принятой технологии работы магазина необходимо отсрочить введение цен в действие, то ввод цен в действие производится менеджером магазина с помощью документа Применение цен номенклатуры.
