Кассовая книга

Документ Кассовая книга (КК) предназначен для формирования и печати листов кассовой книги организации. КК - бухгалтерский регистр для учёта движения наличных денег в кассе. В КК регистрируются все кассовые операции немедленно после их совершения с указанием номера кассового документа, на основании которого произведён приём или выдача наличных денег, фамилии лица, сдавшего или получившего деньги, и суммы операции.

Для включения возможности ведения кассовой книги администратору необходимо установить флаг Кассовая книга (Администрирование – Организации и Финансы) и флаг Использовать в кассовой книге соответствующего элемента справочника Кассы. При этом все документы, созданные кассиром по отмеченной кассе, будут отражаться в кассовой книге.

Список документов можно открыть по ссылке Финансы - Денежные средства - Кассовая книга.

Для удобства работы Пользователь может отфильтровать список листов кассовой книги, установив значение отбора в поле Организация в шапке списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов.

При использовании команды Лист кассовой книги из меню Печать будут распечатаны листы кассовой книги для всех выделенных в списке документов. Контроль полноты кассовой книги осуществляется с помощью отчета Контроль кассовой книги раздела Финансы и в форме списка документов Кассовая книга.

В отчет выводится список документов, не вошедших в кассовую книгу по каким-то причинам. Для включения данных документов в кассовую книгу необходимо перезаполнить табличную часть документа Кассовая книга за дату не вошедшего в книгу ордера, сохранить документ и распечатать новый лист. Также придется перепечатать листы кассовой книги за все последующие даты. Поэтому рекомендуется использовать отчет перед печатью листов кассовой книги.

Новый лист кассовой книги создается с помощью кнопки Создать в командной панели списка документов.

Заполнение шапки документа

В поле Организация пользователь (старший кассир) указывает организацию, от имени которой оформляется документ. Значение поля может быть заполнено по умолчанию в следующих случаях:

В списке листов кассовой книги было установлено значение быстрого отбора по организации;

В настройках текущего пользователя установлено значение организации;

В информационной базе имеется единственная организация.

Значение поля Тип листа определяет печатную форму документа. Пользователь может выбрать следующие значения:

· Обычный - без вывода итогового количества листов;

· Последний в месяце - с выводом итогового количества листов за месяц;

· Последний в году - с выводом итогового количества листов за год.

Заполнение табличной части документа

Чтобы заполнить табличную часть документа, пользователь должен нажать кнопку Заполнить в командной панели табличной части. В результате табличная часть будет заполнена данными о документах Расходный кассовый ордер (РКО) и Приходный кассовый ордер (ПКО) за указанную в шапке документа дату, за исключением документов с операциями Поступление ДС из другой кассы и Выдача ДС в другую кассу.
Поле Корр. счет будет заполнено значением корреспондирующего счета из статьи движения денежных средств, указанной в кассовом документе.
Реквизит Номер листа выводится информативно и пересчитывается при проведении документа.

Печатные формы документа

Печать листов кассовой книги осуществляется по команде Лист кассовой книги выпадающего меню Печать. Печать кассовой книги осуществляется на формате А4. Предусмотрены три шаблона печати, использование которых определяется значением поля Тип листа.
Печать титульного листа и последней страницы осуществляется по команде Титульный лист и последняя страница из меню Печать.
