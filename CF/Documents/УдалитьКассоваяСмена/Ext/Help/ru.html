Кассовые смены

Кассовая смена – период работы кассира за кассовым аппаратом. Этот период работы фиксируется в документе Кассовая смена.

Список документов открывается Продажи – Розничные продажи – Кассовые смены. Для удобства работы Пользователь может отфильтровать список смен по магазину, кассе, статусу кассовой смены выбрав нужные значения в соответствующих полях шапки списка. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные смены, задавая поиск по значениям различных реквизитов.

Новый документ создается автоматически при открытии смены - Продажи – Розничные продажи – Чеки ККМ с помощью кнопки Открыть смену. После выполнения команды система информирует пользователя об открытии смены.

При закрытии смены можно провести автоматическую сверку с данными банка о проведенных за смену платежах. Сверка возможна только тогда, когда используются кассовые аппараты и эквайринговые терминалы с подключением к системе. Если для пользователя установлено дополнительное право: "Проверять непробитые чеки при закрытии смены", то выводиться форма проверки не пробитых чеков, в том числе не пробитых в кассовых документах.

При пробитии чека проверяется допустимость действия пользователя:при попытке пробития чека с оплатой по платежной карте проверяется производилась ли банковская операция, и если нет - то выводится сообщение о невозможности пробития чека. При отмене или аннулировании чека не допускается операция с чеком, успешно оплаченным платежной картой.
