Отчеты о розничных продажах

Документ Отчет о розничных продажах формируется автоматически при оформлении операции Закрытие кассовой смены на основе введенных чеков ККМ по продаже и возврату товаров.

Список отчетов открывается Продажи – Розничные продажи – Отчеты о розничных продажах. Для удобства работы Пользователь может отфильтровать список отчетов по кассе выбрав нужные значения. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные отчеты, задавая поиск по значениям различных реквизитов.

 

Выбранный отчет открывается по двойному клику. На закладке Товары указывается список проданных товаров, а на закладке Возвращенные товары – список товаров, возвращенных в течение кассовой смены с указанием причины возврата товаров - хозяйственной операции. На закладке Оплата платежными картами указывается сумма оплат по различным платежным картам. Предоставленные в качестве подарков товары отображаются на закладке Подарки, продажи по дисконтным картам на закладке Товары (Возвращенные товары) в разрезе номенклатур.

Если производилась продажа товаров в кредит, то в документе появляется еще одна закладка Оплата банковскими кредитами, на которой отображается сумма предоставленных клиентам кредитов при оформлении розничных продаж.

Важно. Чеки ККМ, которые оформлялись за кассовую смену, из информационной базы не удаляются, они сохраняются в информационной базе, но для них устанавливается признак архивации. Такие документы служат только в качестве дополнительной справочной информации, никаких действий с таким документом произвести нельзя.

Новый документ можно добавить с помощью кнопки Создать в командной панели списка. Это может быть сделано, например, в том случае, если необходимо зафиксировать продажи в магазине, в котором не автоматизирован процесс оформления розничных продаж.

На закладке "Прочая выручка" выводиться информация о чеках, пробитых из кассовых ордеров и документов оплаты по эквайрингу. Данные этой закладки учитываются в печатных формах "КМ-6", "КМ-7" (документ "Сводный отчет по кассовой смене").

Отчет о розничных продажах можно вывести по форме Расходная накладная.

Выводятся данные о количестве проданных товаров и услуг, ценах скидках и итоговых суммах продаж.

Список проданных товаров без указания цен и сумм можно также просмотреть в печатной форме Опись номенклатуры. Если для номенклатуры установлено ведение серий, то в графе "Товары" номер серии выводится в скобках после наименования номенклатуры.

Внимание! Продажи слабоалкогольной продукции теперь регистрируются в ЕГАИС не из документа Чек ККМ, а на основании документа Отчет о розничных продажах.

При закрытии смены на основании созданного отчета о розничных продажах автоматически формируется документ Акт списания ЕГАИС из торгового зала на слабоалкогольную продукцию с причиной списания Реализация и передается в ЕГАИС.
