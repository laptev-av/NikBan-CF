Внутреннее перемещение товаров (Ордер на перемещение товаров)

Как правило, это перемещения товаров в торговый зал из складских помещений или перемещение выявленного при продаже брака в складские помещения магазина. Для оформления движения товаров между складами одного магазина служит документ Ордер на перемещение товаров. Никаких настроек для использования ордеров на перемещение не требуется.

Список документов можно открыть по ссылке Склад - Складские операции - Внутреннее перемещение товаров. Для удобства работы Пользователь может отфильтровать список ордеров по магазину, складу-отправителю, складу получателю выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные ордера, задавая поиск по значениям различных реквизитов.

Новый Ордер на перемещение можно добавить с помощью кнопки Создать в командной панели списка. Кроме того, новый документ можно создать на основании документов Поступление товаров, Перемещение товаров и с помощью сервиса Анализ товарного остатка и формирование ордеров.

Заполнение шапки документа

В шапке нового документа заполняются следующие поля:

· Магазин, в котором происходит перемещение;

· Склад-отправитель, с которого будет отгружен товар;

· Склад-получатель, на который будет принят товар;

· Организация.

Некоторые из этих полей заполняются автоматически в следующих случаях:

· соответствующие значения заданы в настройках пользователя;

· перед созданием нового ордера в списке ордеров были установлены значения отбора;

· новый ордер был создан на основании документов Поступление товаров или Перемещение товаров, в этом случае значения магазина и склада-отправителя загружаются из документов-оснований;

· новый ордер был создан с помощью сервиса Анализ товарного остатка и формирование ордеров.

Заполнение табличной части

При создании ордера на основании документов Поступление товаров или Перемещение товаров данные в табличную часть загружаются из документов-оснований.

Табличная часть документа может быть заполнена автоматически по результатам выполнения сервиса Анализ товарного остатка и формирование ордеров.

Добавить товары в табличную часть можно следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· загрузить из терминала сбора данных;

· с помощью сервиса Подобрать товары;

· по штрихкоду.

Добавление товара в табличную часть по штрихкоду

Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне.

Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1.

Заполнение цен товаров

Цены на товар вводятся информационно для контроля розничных цен в торговом зале магазина. При создании документа и добавлении товаров в табличную часть цены автоматически заполняются значениями текущих розничных цен магазина.

Если ордер был создан на основании документов Поступление товаров или Перемещение товаров, то можно воспользоваться сервисом Заполнить по основанию из меню Цены в командной панели табличной части. Для того, чтобы перезаполнить цены значениями текущих розничных цен можно воспользоваться сервисом Заполнить по розничным ценам магазина.

Печатные формы документа

Из меню Печать командной панели можно сформировать печатную форму документа Ордер на перемещение товаров, например, для того, чтобы передать его по факсу на соответствующий склад.

Список товаров без указания цен и сумм можно также просмотреть в печатной форме Опись номенклатуры.
