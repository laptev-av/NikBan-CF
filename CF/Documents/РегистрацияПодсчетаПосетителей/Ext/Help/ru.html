Подсчеты посетителей

При загрузке данных из системы подсчета количества посетителей система создает документ "Регистрация подсчета посетителей".

Список документов открывается Маркетинг – Посетители - Подсчеты посетителей. Для удобства работы Пользователь может отфильтровать список по магазину, счетчику, выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов.

В поле Счетчик подсчета посетителей записывается ссылка на справочник Счетчики подсчета посетителей. В табличной части "Данные подсчета посетителей" - количество вошедших и вышедших из торгового зала посетителей за определенное время - время регистрации. В табличной части "Данные подсчета сотрудников" записываются Количество проходов сотрудников в соответствии с данными, определенными для данного счетчика.

Итоговый показатель Количество посетителей равен ("Количество входящих" + "Количество выходящих" - "Количество проходов сотрудников") / 2.

Новый документ можно добавить с помощью кнопки Создать в командной панели списка.

В открывшейся форме заполнить магазин и выбрать из справочника нужный счетчик проходов. В табличной части документа необходимо внести время регистрации данных и количество вошедших и вышедших посетителей с момента предыдущей записи табличной части. Например, если установлена частота регистрации - один час с 9.00 до 21.00, то нужно создать 12 записей табличной части.

На закладке Сотрудники можно зарегистрировать количество проходов сотрудников магазина, если они не были учтены при подсчете посетителей.
