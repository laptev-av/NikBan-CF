Отчет Комитенту о продажах.

К категории комиссионных относятся сделки, в которых один из участников соглашения представляет интересы третьей стороны, получая за свою работу материальное вознаграждение. Соответственно комитентом (принципалом) называется сторона, от чьего имени действует лицо, и дающая своему представителю указания на совершение тех или иных действий.

Данный документ необходим для отображения информации о получаемом вознаграждении.

Список документов Отчет Комитенту о продажах открывается по ссылке Закупки - Комиссионная торговля – Отчеты комитенту. Для удобства работы Пользователь может отфильтровать список по комитент, магазину, кассе, организации, ответственный выбрав нужные значения в соответствующих полях шапки списка.

Выбранный документ может быть напечатан по форме:

· Акт об оказании услуг

· Отчет комитенту о продажах

и/или оформить счет-фактуру на вознаграждение.

Новую запись Отчеты комитенту о продажах можно добавить с помощью кнопки Создать (Отчет Комитенту о продажах) в командной панели списка. Вначале необходимо заполнить шапку документа, а затем добавить товары с помощью команды Добавить новый элемент, Найти по штрихкоду/магнитному коду или с помощью сервиса Заполнить (Заполнить по результатам продаж/Подобрать товары по результатам продаж).

На основании документа документа можно создать:

· Расходный кассовый ордер;

· Приходный кассовый ордер;

· Регистрация безналичной оплаты;

· Счет-фактура" (выданный) на вознаграждение.
