Ассортимент (изменение ассортимента)

Документ отражает факт изменения ассортимента указанного формата магазинов.

Список документов открывается Маркетинг – Ассортимент - Ассортимент. Для удобства работы Пользователь может отфильтровать список документов по формату магазина выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов. На основании выбранного документа можно создать:

· Ордер на отражение недостач товаров,

· Перемещение товаров,

· Списание товаров,

· Установка цен номенклатуры

Выбранный документ можно напечатать по форме Переоценка товаров.

Новый документ можно добавить с помощью кнопки Создать в командной панели списка.

Ассортимент состоит из категорий товаров, которые будут представлены в данном магазине, и определяется форматом магазина:вместимостью торгового зала, местоположением и целевой группой покупателей.

При создании нового документа пользователь заполняет основные реквизиты:

Дата изменения;

Формат магазина – один из определенных пользователем;

Товарные позиции, которые будут представлять указанные в документе Квоты ассортимента категории в данном формате;

Вид цен – вид цены розничной продажи в данном формате по умолчанию определяется по правилу ценообразования для данного формата магазина.

Таким образом, пользователь в одном документе указывает, какие, в каких магазинах, когда и по какой цене будут продаваться товары.

Поля Операция и Стадия данного документа определяют этап жизненного цикла товаров в ассортименте каждого формата магазина и соответствующие этому этапу ограничения операций товарообращения.

Заполнение табличной части

Заполнить табличную часть документа номенклатурой можно вручную с помощью кнопки Добавить или с помощью специальных подборов из меню Подбор и с помощью сервисов из меню Заполнить.

Сервис Подбор

Открыть подбор по категориям вызывается командой меню Подбор табличной части документа Изменение ассортимента. Подбор по категориям позволяет просматривать номенклатуры в разрезе существующих товарных категорий.

В левой части окна выводится справочник Категории товаров с указанием общего числа номенклатурных позиций в данной категории, их текущей наполненности для данного формата и квот товаров. В колонке Всего выводится число существующих в базе номенклатурных наименований, относящихся к данной категории; в колонке Наполненность – число номенклатурных наименований, которые есть в ассортименте магазина данного формата; в столбце Наполненность + подобрано – число номенклатурных наименований, еще не включенных в заполняемый документ Ассортимент, но включенных в ассортимент магазина данного формата.

В правой части окна выводятся номенклатурные единицы, относящиеся к выбранной категории товаров. Двойным щелчком левой кнопки мыши товар добавляется в список подобранных позиций в нижней части экрана. Если товар уже входит в ассортимент данного формата, система выдаст предупреждение о том, что добавление данного товара не требуется. Список отобранных товаров выводится в нижней табличной части, если установлен флажок Показывать подобранные товары. Пользователь может контролировать промежуточные результаты своей работы по значению в колонке Наполнено + подобрано в сравнении с установленной для данной категории квотой. Для завершения процедуры отбора и добавления подобранных позиций в ассортимент необходимо нажать команду Перенести в документ.

Открыть подбор по отбору вызывается командой меню Подбор табличной части документа Изменение ассортимента.

Отбор может осуществляться по доступным для отбора полям, которые добавляются в правую таблицу отбора по команде Выбрать. Когда все условия отбора заданы, необходимо нажать кнопку Заполнить, после чего в нижней части документа заполнится таблица Подобранные товары в соответствии с заданными условиями. Чтобы выбранные товары отобразились в документе Ассортимент, необходимо выполнить команду Перенести в документ.

Пример. Маркетологу необходимо перенести в архив все выведенные из ассортимента молочные продукты в продуктовых магазинах. Для этого необходимо выбрать формат магазина «Продукты». Вид номенклатуры установить равным «Молочные продукты», в поле Разрешены закупки установить значение Нет, а в поле Разрешены продажи – Да и изменить логику сочетания последних двух параметров с ИЛИ на И.

Открыть подбор вызывается командой меню Подбор табличной части документа Изменение ассортимента.

В окне навигации доступен справочник, в котором можно выбрать необходимый вид номенклатуры. Соответствующие текущему виду номенклатуры товары выводятся в правой части экрана. Двойным щелчком левой кнопки мыши товар можно добавить в список подобранных. После того как отбор завершен, необходимо нажать Перенести в документ. В результате товары появятся в редактируемом документе Ассортимент.

Сервис Заполнить

· Роль для всех товаров. Выбор из трех опций: постоянный ассортимент/дополнительный ассортимент/пробный ассортимент.

· Вид цен для всех товаров. Выбор из пяти опций: розничная/мелкооптовая/цена распродажи/услуги в магазинах/минимальная цена продажи в розницу.

· Заполнить по ассортименту формата. Таблица заполняется теми товарами, которые уже были добавлены в ассортимент магазинов данного формата в ранее проведенных документах Ассортимент товаров;

· Заполнить по складским остаткам и оборотам. Таблица заполняется товарами, по которым существовало движение за указанный период.

При заполнении с помощью сервисов табличная часть по желанию пользователя может предварительно быть очищенной, а может быть дополненной новыми значениями. Способ выбирается в открывающемся при автоматическом заполнении диалоговом окне.
