Заказы на перемещение товаров

Документ Заказ на перемещение служит для фиксирования потребности в перемещении товаров из распределительного центра управляющей системы или из одного магазина в другой.

Список документов можно открыть по ссылке Закупки - Снабжение - Заказы на перемещение. На основании этого документа по факту перемещения товаров может быть оформлен документ Перемещение товаров. Предусмотрена печатная форма Заказа на перемещение, которую можно использовать, например, для передачи заказа по факсу.

Новый документ Заказ на перемещение можно сформировать вручную или с помощью сервиса Анализ продаж и формирование заказов из панели действий раздела Запасы и закупки.

Заполнение шапки документа

В шапке документа указывается Магазин-заказчик, на который требуется переместить товар, Магазин-поставщик или распределительный центр (магазин с типом объекта розничной сети «Склад управляющей системы»), Желаемая дата поступления товаров, Длительность перемещения. Если разность между желаемой датой поступления и датой заказа меньше предполагаемой длительности перемещения, то при попытке проведения заказа на экран будет выведено предупреждение об ошибке.

Добавление товаров в табличную часть

Строки в табличную часть документа можно добавить следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· с помощью сервиса Подобрать товары;

· по штрихкоду. Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне. Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1.

Закрытие Заказа на перемещение

При оформлении документа Перемещение товаров на основании документа Заказ на перемещение, если заказ выполнен полностью, то в документе-основании устанавливается статус Закрыт. Частично выполненный заказ на перемещение можно закрыть вручную, установив статус Закрыт в шапке документа. Можно настроить запуск регламентного задания Закрытие заказов на перемещение, которое будет автоматически закрывать просроченные заказы. Регламентные задания настраиваются на странице регламентных заданий, которую можно открыть по ссылке Администрирование - Обслуживание - Регламентные и фоновые задания.
