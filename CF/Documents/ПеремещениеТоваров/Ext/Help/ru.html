Перемещение товаров

Для оформления перемещения товаров между магазинами создается документ Перемещение товаров. В частности, с помощью этого документа можно оформить возврат товара из магазина на распределительный склад. Оформление перемещения товаров всегда производится по ордерной схеме.

При совместной работе с конфигурацией «Управление торговлей» поддерживается основной сценарий централизованного снабжения товарами магазинов сети из распределительного центра. Поступление товаров в магазин оформляется с помощью документов Перемещение товаров, которые оформляются в управляющей системе и мигрируют в конфигурацию «Розница». Помимо этого, сценария, в процессе реализации товаров периодически может возникать потребность в перераспределении товаров между магазинами. В этом случае менеджер центрального офиса или магазина на основании проведенного анализа продаж и остатков оформляет документы перемещения товаров из одного магазина в другой.

Список документов перемещение можно открыть по ссылке Закупки - Снабжение - Перемещения товаров. Для удобства работы Пользователь может отфильтровать список документов по магазину отправителю/получателю, складу отправителю/получателю выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов.

Новый документ может быть создан на основании документа Заказ на перемещение или непосредственно с помощью кнопки Создать в командной панели списка документов перемещения товаров.

Заполнение документа Перемещение товаров

Если документ Перемещение товаров создается на основании документа Заказ на перемещение, то реквизиты шапки документа и товарные строки заполняются из документа-основания, и формируется информационная ссылка на документ-основание.

При непосредственном создании нового документа Перемещение товаров нужно заполнить поля шапки документа: Магазин отправитель, Склад отправитель, Магазин получатель и Склад получатель. По умолчанию эти поля заполняются в соответствии с настройками Пользователя, а если они не установлены, то значениями полей отбора из списка документов. Если справочник содержит единственный элемент, то он будет подставлен по умолчанию.

Комиссионная торговля

1. На вкладке "Дополнительно" заполняют реквизиты "Контрагент" и "Договор", того комитент чей товар хотят переместить на другой склад.

2. Документ делает движения по регистру накопления "Товары организаций" с заполнением измерений "Поставщик" и "Договор"

Добавление товаров в табличную часть

Товары в табличную часть документа можно добавить следующими способами:

· с помощью кнопки Добавить командной панели табличной части документа;

· с помощью сервиса Подобрать товары;

· по штрихкоду. Чтобы добавить товар по штриховому коду, нужно перейти в табличную часть и считать штриховой код сканером. Кроме того, можно нажать кнопку Поиск по штрихкоду в командной панели или горячую клавишу F7 и ввести символы штрихкода вручную или с помощью сканера в открывшемся окне. Если данный штрихкод зарегистрирован в системе, то соответствующий ему товар будет добавлен в табличную часть. В противном случае в правой части формы появится сообщение об ошибке. Если товар с таким штрихкодом уже введен в табличную часть, то строка с данным товаром будет выделена, а количество увеличится на 1.

Редактирование данных в строке табличной части

В каждой товарной строке заполняются данные в следующих полях:

· Характеристика номенклатуры - можно выбрать из списка характеристик данной номенклатуры;

· Упаковка - по умолчанию указывается основная единица измерения, но может быть выбрана любая упаковка данной номенклатуры;

· Количество - количество поступивших единиц и упаковок;

· Цена - может быть заполнена при помощи сервиса заполнения цен и при необходимости изменена вручную;

· Сумма - рассчитывается на основании введенных данных.

Сервисы заполнения цен товаров

Цены в строках документа можно ввести вручную или воспользоваться сервисами заполнения Цены в командной панели табличной части документа:

· Заполнить цены по розничным ценам - цены перемещения будут установлены по розничным ценам магазина-отправителя во всех строках таблицы;

· Заполнить цены выделенных строк по розничным ценам - будут заполнены цены только в выделенных пользователем строках;

· Заполнить цены по виду цен - цены перемещения будут установлены по выбранному пользователем виду цен, например, специальные трансфертные цены для перемещения между магазинами, для всех строк таблицы;

· Заполнить цены выделенных строк по виду цен - по выбранному виду цен будут заполнены цены только в выделенных пользователем строках.

Распределение товаров

Если отгрузка товара производится с разных складов Магазина отправителя и/или приемка товара производится на разные склады Магазина получателя, то можно воспользоваться сервисами Распределить отгрузку по складам и/или Распределить приемку по складам, которые запускаются из выпадающего списка Распределение товаров в командной панели документа.

Документы, которые можно создать на основании документа Перемещение товаров

На основании документа Перемещение товаров можно создать следующие документы:

· Расходный ордер - документ, уменьшающий складские остатки;

· Приходный ордер - документ, увеличивающий складские остатки;

· Ордер на перемещение товаров – документ для оформления движения товаров между складами одного магазина;

· Передача товаров – документ для оформления передачи товаров в другую организацию предприятия.

Реквизиты создаваемых документов заполняются из выбранного пользователем в качестве основания документа Перемещение товаров.

Печатные формы документа Перемещение товаров

Для документа предусмотрены следующие печатные формы, доступные в выпадающем списке Печать:

· 1-Т (Товарно-транспортная накладная),

· М-11 (Требование-накладная),

· М-4 (Приходный ордер),

· Накладная на перемещение,

· ТОРГ-13 (Накладная на внутреннее перемещение),

· Транспортная накладная.

· Опись номенклатуры (список товаров без указания цен и сумм).

Кроме того, можно распечатать Этикетки и Ценники.
