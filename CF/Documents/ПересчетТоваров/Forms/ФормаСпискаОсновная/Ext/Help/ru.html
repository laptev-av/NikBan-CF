Пересчеты товаров

Документ Пересчет товаров предназначен для фиксирования результатов проведенной на складе инвентаризации.

Документ Пересчет товаров может быть создан только на основании документа Приказ на пересчет товаров. На основании одного приказа может быть создано несколько документов пересчета.

Документ Пересчет товаров не изменяет учетное количество товаров. Если в результате пересчета товаров выявлены излишки или недостачи, то для того, чтобы скорректировать данные управленческого учета, нужно на основании исходного документа Приказ на пересчет товаров создать документы Пересортица товаров, Списание товаров, Оприходование товаров. Кроме того, если в магазине принята ордерная схема оформления инвентаризации, то для того, чтобы привести учетное количество к фактическому нужно на основании исходного документа Приказ на пересчет товаров создать документ(ы) Ордер на отражение результатов пересчетов товаров.

Список документов пересчета товаров можно открыть по ссылке Склад - Инвентаризация - Пересчеты товаров.

Для удобства работы Пользователь может установить быстрый отбор по полям Магазин, Склад и Приказ в шапке списка документов. Кроме того, с помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов.
