Ввод остатков денежных средств

Документ "Ввод остатков денежных средств" предназначен для ввода начальных остатков денежных средств в кассах магазина.

Список документов открывается Администрирование - Первоначальное заполнение ИБ - Ввод остатков денежных средств.

С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Документ Ввод остатков денежных средств создается с помощью нажатия кнопки Создать.

В табличную часть документа вводится информация о каждой кассе, в которой имеется остаток денежных средств, и сумме денежных средств в каждой кассе.
