Приказ на пересчет товаров (инвентаризацию)

Документ Приказ на пересчет товаров является основанием для выполнения пересчетов товаров и последующего оформления выявленных расхождений между фактическим наличием товаров и учетными данными.

Список документов можно открыть по ссылке Склад - Инвентаризация - Приказы на пересчет товаров. Для удобства работы Пользователь может отфильтровать список приказов по магазину, складу, состоянию выбрав нужные значения в соответствующих полях шапки списка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов. На основании документа Приказ на пересчет товаров можно оформить документы:

· Пересчет товаров - пересчет товаров на основании приказа должен быть проведен, иначе приказ останется невыполненным. На основании одного приказа может быть создано несколько документов пересчета.

· Пересортица товаров - если в результате пересчета выявлены равные по количеству излишки и недостачи товаров с близкими параметрами, то можно оформить пересортицу.

· Списание товаров - если в результате пересчета выявлена недостача товара, а также бракованный или испорченный товар, то нужно оформить списание.

· Оприходование товаров - если в результате пересчета выявлен излишек товаров, то его нужно зарегистрировать.

· Ордер на отражение результатов пересчетов товаров - если инвентаризация проводится по ордерной схеме, то после пересчета сотрудники магазинов приводят учетные остатки товаров к фактическим с помощью ордеров. В этом случае документы управленческого учета Пересортица товаров, Списание товаров, Оприходование товаров оформляются сотрудниками офиса и не влияют на количественные остатки.

На основании документа Приказ на пересчет товаров можно напечатать документы: регламентные формы ИНВ-3 Инвентаризационная опись товаров и ИНВ-19 Сличительная ведомость. Эти формы будут заполнены данными из всех документов пересчета, созданных на основании выбранного приказа.

Новый документ Приказ на пересчет товаров можно создать с помощью одноименной ссылки из панели действий раздела Склад или из списка документов с помощью кнопки Создать в командной панели списка.

Заполнение полей документа

При создании документа Приказ на пересчет товаров некоторые поля могут быть автоматически заполнены значениями, установленными для пользователя по умолчанию. Пользователь редактирует или заполняет следующие поля:

· Магазин и Склад магазина - место проведения пересчета товаров;

· Организация - заполняется по организации, указанной в настройках Склада;

· Период инвентаризации, в течение которого нужно произвести пересчет товаров по данному приказу;

Правила отбора товаров - при проведении полной инвентаризации следует выбрать предустановленное правило Полная инвентаризация. При проведении частичной инвентаризации следует выбрать соответствующее правило из списка правил, настроенных заранее. После выбора правила отбора товаров в окне создания приказа появляется текстовая строка с описанием параметров выбранного правила. Если на основании данного приказа предполагается создание нескольких документов пересчета товаров, то в правиле отбора рекомендуется установить флажок Не выводить пересчитанные товары данного приказа. В этом случае каждый следующий создаваемый документ Пересчет товаров будет содержать набор уникальных товаров, которые не были пересчитаны ни в одном из ранее созданных документов на пересчет товаров для данного приказа. Таким образом, в рамках одной инвентаризации товары могут быть поделены на не пересекающиеся между собой группы.

Поле Статус управляется пользователем вручную и имеет только два значения:

· В работе – устанавливается в момент создания документа;

· Пересчет завершен – устанавливается в момент завершения всех пересчетов по данному приказу.

Сервисы документа

Если в результате пересчета товаров на основании данного приказа были выявлены товары к списанию и, соответственно, был создан документ Списание товаров, то можно воспользоваться помощником заполнения причин списания, который запускается по команде Заполнить причины списания из меню Сервис командной панели документа Приказ на пересчет товаров. В открывшемся окне можно назначить необходимые значения поля Аналитика хозяйственной операции для товаров, подлежащих списанию по текущему приказу.

Если в результате пересчета товаров на основании данного приказа нужно оформить документ Пересортица товаров, то можно воспользоваться помощником создания такого документа, который запускается по команде Зачет пересортицы из меню Сервис командной панели документа Приказ на пересчет товаров.

Переходы к связной информации

Из панели навигации документа по ссылке Оформление можно сформировать отчет Оформление излишков/недостач товаров. В этот отчет выводится информация о количестве обнаруженных при пересчете по данному приказу излишков и недостач товаров, на которые еще не оформлены ордера или документы управленческого учета.
