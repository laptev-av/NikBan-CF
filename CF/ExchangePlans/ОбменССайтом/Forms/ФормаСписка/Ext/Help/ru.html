Обмен с сайтами

Позволяет настроить обмен данными между учетной информационной системой и Интернет-магазином, поддерживающим протокол обмена между системой "1С:Предприятие" и сайтом.

Основные возможности

  • создание нового сайта в домене 1С-UMI;
  • выгрузка из программы на сайт каталога товаров, включая все свойства товаров, цены и остатки товаров на складах, автоматическое поддержание этих данных в актуальном состоянии;
  • загрузка в программу заказов, оформленных в Интернет-магазине, обновление данных на сайте о статусах оплаты и отгрузке по заказам.

Форма отображает список настроеных узлов обменов с сайтом.

Выводимые в список данные

Доступные команды

Выводимые в список данные
  • Наименование — имя узла;
  • Дата загрузки — дата последней загрузки заказов с сайта;
  • Дата успешной загрузки — дата последней успешной загрузки заказов с сайта;
  • Дата выгрузки — дата последней выгрузки товаров на сайт или обновления информации о заказах;
  • Дата успешной выгрузки — дата последней успешной выгрузки товаров на сайт или обновления информации о заказах;
  • Обмен товарами — признак выгрузки на сайт каталога товаров;
  • Обмен заказами — признак загрузки с сайта заказов покупателя.

Доступные команды
  • Выполнить обмен — запускает сеанс обмена данными с сайтом;
  • Выполнить полный обмен — в данном режиме обмена данные каталога товаров перезапишут существующий каталог (если он уже был выгружен на сайт);
  • Показать зарегистрированные изменения — открывает форму просмотра товаров и заказов, данные о которых будут выгружены на сайт.

Команды Выполнить полный обмен данными и Показать зарегистированные изменения доступны из списка Еще.
