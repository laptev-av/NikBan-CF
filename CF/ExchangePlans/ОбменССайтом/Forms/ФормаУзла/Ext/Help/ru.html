Узел обмена с сайтом

Позволяет настроить обмен данными между учетной информационной системой и Интернет-магазином, поддерживающим протокол обмена между системой "1С:Предприятие" и сайтом.

Основные возможности

  • создание нового сайта в домене 1С-UMI;
  • выгрузка из программы на сайт каталога товаров, включая все свойства товаров, цены и остатки товаров на складах, автоматическое поддержание этих данных в актуальном состоянии;
  • загрузка в программу заказов, оформленных в Интернет-магазине, обновление данных на сайте о статусах оплаты и отгрузке по заказам.

Основные настройки
  • Режим обмена данными — определяет, какие наборы данных будут участвовать в обмене (выгрузка товаров и/или обмен информацией о заказах);
  • Способ обмена данными — прямой обмен с сайтом или посредством каталога на диске, в который будут помещаться файлы обмена;
  • Автоматический обмен — возможность включить автоматический обмен и настроить периодичность его выполнения;
  • Расширенная настройка — настройка размера порции выгружаемых на сайт данных и количества попыток повтора обмена, в случае возникновения ошибки.

Настройка выгрузки товаров
  • Организация - владелец каталога товаров — организация, от имени которой публикуется каталог товаров на сайте;
  • Использовать узел обмена - определяет использование режима оптимизации размера выгружаемых данных. Если использовать режим Обновление на сайте цен и остатков товаров, то будут выгружаться данные об уже выгруженных на сайт товарах (цены и остатки). Новые товары и виды цен выгружаться не будут.
  • Выгружаемые данные — каталог товаров и/или цены и остатки. Указывается необходимость выгрузки изображений и прочих файлов, складов, доступных покупателям для выбора на сайте. Также возможно установить вариант классификации номенклатуры — по видам номенклатуры. По умолчанию вариант классификации - по группам номенклатуры;
  • Режим выгрузки данных — полная выгрузка, или только изменившиеся с последней выгрузки данные;
  • Настройки каталогов товаров — необходимо определить один или несколько каталогов на сайте и настроить отбор выгружаемых товаров и услуг.

Выгрузка только цен по соглашениям и остатков по складам без каталога товаров имеет смысл, если каталог товаров ранее уже был выгружен на сайт.

Обмен заказами
  • Параметры поиска контрагентов — указывается способ поиска контрагентов.
  • Единица измерения новой номенклатуры — единица измерения для номенклатуры, которая будет загружена с сайта.
  • Группа для новых контрагентов — указывается группа, в которую будут записаны новые контрагенты, загруженные с сайта.
  • Группа для новой номенклатуры — указывается группа, в которую будет записана новая номенклатура, загруженная с сайта.
  • Организация — организация, на которую оформляется заказ, загруженный с сайта.
  • Номер заказа на сайте — свойство заказа, которым определяется номер заказа на сайте.
  • Дата заказа на сайте — свойство заказа, которым определяется дата заказа на сайте.
  • Соответствие статусов заказов — устанавливает соответствие между статусами заказа на сайте и статусами заказа в 1С, чтобы автоматически устанавливать статусы у загружаемых документов.

Переход на сайт

Содержит ссылки для быстрого перехода на сайт и в панель администрирования сайта.

Команды
  • Выполнить обмен — запускает процедуру обмена данными по настройкам, заданным в узле.
  • Выполнить полный обмен данными — запускает процедуру обмена данными по настройкам, заданным в узле, за исключением настройки Режим выгрузки данных.
  • Показать зарегистрированные изменения — показывает список объектов, зарегистрированных к выгрузке, по которым будет производиться отбор при выполнении обмена данными в режиме выгрузки данных Изменения.
