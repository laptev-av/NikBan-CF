Оборот наличных ДС по статьям ДДС

В отчете показывается информация движении наличных денежных средств в разрезе организаций и статей движения денежных средств. Отчет имеет два варианта настройки:

Оборот наличных ДС по статьям ДДС за период - настройку удобно использовать для просмотра оборотов за короткий интервал времени, определенный датами.

Оборот наличных ДС по статьям ДДС за период (по месяцам) - удобно использовать для просмотра оборотов за длительный промежуток времени. Обороты суммируются по месяцам и выводятся в колонки отдельно для приходных и расходных операций за каждый месяц, вошедший в период отчета. Отчет открывается по ссылке Финансы – Отчеты по финансам – Денежные средства – Оборот наличных ДС по статьям ДДС.
