Остатки бонусных баллов

В отчет Остатки бонусных баллов выводится информация о накопленных баллах, о количестве баллов, подлежащих списанию, т.е. не использованных вовремя, и о доступном к оплате остатке. Отчет можно открыть по ссылке Маркетинг - Отчеты по маркетингу – Маркетинговые акции - Бонусные программы - Остатки бонусных баллов.

Пользователь может установить быстрый отбор по бонусной программе лояльности и/или дисконтной карте. Дополнительные карты, по которым накопления относятся на основную карту для накоплений, в отчете не отображаются. Все данные выводятся по основной карте для накоплений.

Следует учитывать, что дополнительные карты, по которым накопления относятся на основную карту для накоплений, в отчете не отобразятся. Все данные выведутся по основной карте для накоплений.
