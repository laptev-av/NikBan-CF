Продажи по дисконтным картам чека

В отчете Продажи по дисконтным картам чека ККМ производится детальный анализ сумм и скидок по каждому чеку, по которому была предоставлена скидка по дисконтной карте. Отчет можно открыть по ссылке Продажи - Отчеты по продажам – Розничные продажи - Продажи по дисконтным картам чека.

 

Настройки отчета

· Период – при выставленном флажке отчёт будет строиться за указанный период, иначе за все время.

· Первое поле – название периода (день, неделя, декада, квартал, полугодие, год, произвольный (с - по))

· Второе поле – конкретные значения периода

· Магазин – магазин, по которому вас интересует отчет.

· Номенклатура – номенклатура, по которой вас интересует отчет.

В отчет выводятся показатели:

· Количество чеков ККМ – количество чеков ККМ, пробитых по данной дисконтной карте за указанный период;

· Количество чеков на номенклатуру – количество различных номенклатурных позиций, пробитых по данной карте за указанный период;

· Количество товаров – количество данных номенклатурных позиций, купленных по карте за период;

· Сумма со скидкой;

· Сумма скидки;

· Сумма без скидки;

· Процент скидки – сумма скидки, отнесенная к сумме продаж без скидки, в процентах.

· По умолчанию в отчет выводятся строки номенклатуры с учетом характеристики и единицы измерения, сгруппированные по магазину и дисконтной карте. Для изменения настроек нужно нажать кнопку Настройка.
