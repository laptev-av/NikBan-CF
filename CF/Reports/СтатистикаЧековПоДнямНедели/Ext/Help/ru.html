Статистика чеков по дням недели

Отчет производит анализ загруженности касс в рабочие и выходные дни. Отчет открывается по ссылке Продажи – Отчеты по продажам – Розничные продажи - Статистика чеков по дням недели. Для корректной работы отчета необходимо, чтобы в периоде отчета было выбрано целое количество недель. (Дата начала всегда должна быть понедельник, дата окончания – воскресенье.)

В отчете показывается среднее количество чеков, пробитое кассиром в течение дня.

Среднее количество чеков по кассе в день. Рассчитывается по формуле: Кол-во чеков, пробитых за этот день по данной кассе за период / Количество дней, которое отработала касса за указанный период.
Среднее количество чеков по кассе за неделю. Рассчитывается по формуле: Кол-во чеков, пробитых за неделю по данной кассе за период / Количество дней, которое отработала касса за указанный период в течении недели.
Среднее количество чеков по магазину в день. Рассчитывается по формуле: Количество чеков в этот день по магазину за период / Количество касс, работавших в этот день в магазине.
Среднее количество чеков по магазину за неделю. Рассчитывается по формуле: Количество чеков, пробитых за всю неделю по данному магазину / Количество проработанных дней всеми кассами магазина.

В группировке "По магазину" на каждый день недели выводятся дополнительные показатели

Среднее количество посетителей. Рассчитывается по формуле: Количество посетителей = ("Количество входящих" + "Количество выходящих" - "Количество проходов сотрудников") / 2.

Коэффициент конвертации = Среднее количество чеков ККМ / Среднее количество посетителей.

Примечание
Касса считается работающей, если по ней был пробит хотя бы один чек в течение дня.
