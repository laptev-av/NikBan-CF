Конвертация посетителей

Одним из важнейших показателей эффективности работы магазина является Коэффициент конвертации - показатель удовлетворенности спроса покупателей.

Отчет можно открыть по ссылке Маркетинг – Отчеты по маркетингу – Посетители – Конвертация посетителей.

Отчет допускает группировки: Магазин, Период: день, неделя, декада, месяц, квартал, полугодие, год.

Данные в отчет выводятся в разрезе магазинов за установленный пользователем период:

· Количество чеков продажи – общее количество чеков ККМ, зарегистрированных в данном магазине за отчетный период;

· Количество посетителей – показатель рассчитывается по формуле: (Количество входящих + Количество выходящих - Количество проходов сотрудников) / 2;

· Коэффициент конвертации – рассчитывается по формуле: Количество покупателей магазина / Количество посетителей;

· показатель Коэффициент конвертации также добавлен в отчеты Статистика чеков по дням недели и Статистика чеков ККМ в разрезе рабочего времени (интервал в часах).

Отчет строится на данных регистра накоплений "Посетители", в котором за время регистрации записывается количество вошедших и вышедших из торгового зала посетителей, и Количество проходов сотрудников в соответствии с данными, определенными для данного счетчика.

Количество посетителей = ("Количество входящих" + "Количество выходящих" - "Количество проходов сотрудников") / 2.
