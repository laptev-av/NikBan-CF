Действующие маркетинговые акции.

Отчет отображает все маркетинговые акции, действующие на указанную дату. Открывается по ссылке Маркетинг – Отчеты по маркетингу – Маркетинговые акции - Действующие маркетинговые акции.

Пользователь может установить отбор по магазину, а также с помощью редактирования отбора, с помощью команды Настройки.

Редактирование отбора

Для того, чтобы создать условие отбора выберите поле в списке Доступные поля, установите вид сравнения отбора в колонке Вид сравнения, и укажите Значение для сравнения. Для редактирования последовательности элементов отбора используйте стрелки командной панели.
