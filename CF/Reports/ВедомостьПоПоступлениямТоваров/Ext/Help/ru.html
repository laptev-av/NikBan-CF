Ведомость по поступлениям товаров

Отчет отображает сведения об объемах поступившего товара в разрезе поставщиков. Отчет открывается по ссылке Закупки - Отчеты по запасам и закупкам - Закупки - Ведомость по поступлениям товаров.

Возможно настройка группировки по периодам. Для вывода развертки периодов в столбцах необходимо использовать расширенную настройку отчетов. На закладке Структура выполнить настройки, как показано ниже:

Сохранить вариант отчета под новым именем. В сформированном отчете данные по установленному периоду выведутся в колонках.

При настройке произвольного периода отчета теперь можно не вводить конечную дату. В отчет выведутся все зарегистрированные данные.
