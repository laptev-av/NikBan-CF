Анализ причин возвратов поставщикам

Отчет Анализ причин возвратов поставщику предназначен для оценки возвратов товаров относительно поступлений этих товаров в разрезе причин возвратов. Отчет открывается по ссылке Закупки - Отчеты по запасам и закупкам - Работа с поставщиками - Анализ причин возвратов поставщикам.

В отчете выводится количество возвращенных поставщику позиций номенклатуры в сравнении с поступившим за тот же период количеством. Для каждой номенклатурной позиции отображаются также текущая розничная цена и суммовые показатели по количеству и по цене поступления. Строки группируются в разрезе причин возврата товаров поставщику. Для регистрации причин возврата используется справочник Аналитики хозяйственных операций, в котором для хозяйственной операции Возврат поставщику могут быть созданы несколько аналитик. Справочник Аналитики хозяйственных операций заполняет администратор на странице Запасы и закупки раздела Администрирование.
