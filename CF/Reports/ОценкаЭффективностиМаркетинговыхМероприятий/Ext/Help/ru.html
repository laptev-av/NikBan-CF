Оценка эффективности маркетинговых мероприятий

Отчет, показывающий изменения основных показателей продаж по выбранной акции. Отчет формируется в разрезе участвующих в акции магазинов. Отчет открывается по ссылке Маркетинг – Отчеты по маркетингу – Маркетинговые акции - Оценка эффективности маркетинговых мероприятий.

В отчет выводятся следующие показатели:

· Сумма продаж по акционным товарам;

· Себестоимость продаж по акционным товарам;

· Прибыль, полученную при продаже акционного товара;

· Сумма скидки, предоставленную при продаже товара по акции;

· Остатки - оценка стоимости акционных товаров по себестоимости;

· Количество чеков, содержащих акционный товар;

· Средняя сумма чеков, содержащих акционный товар.

Показатели подсчитываются только по товарам, указанным в сегментах скидок для данной акции, или по всем товарам, если явного указания нет, и выводятся в три колонки:до акции, за период действия акции (во второй колонке) и изменение показателя во время акции относительно первоначального уровня в процентах.

Отчет позволяет сделать расшифровки показателей, сформированные с учетом отбора и по тем же периодам, что и выведенный на экран общий отчет:

· Отчет Оценка продвижения товаров открывается при щелчке на поле "Сумма продаж", "Себестоимость продаж", "Прибыль"

· Отчет Оценка применения скидок - для поля "Сумма скидки"

· Отчет Оценка распродажи - для поля "Остатки"

· Отчет Оценка продаж - для полей "Количество чеков" и "Средняя сумма чеков".
