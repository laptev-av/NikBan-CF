Варианты подбора товара

Подбор позиций товара производится с помощью команды Подобрать товары в командной панели табличной части документа. Пользователь может установить вариант подбора:

· по иерархии номенклатуры;

· по виду номенклатуры;

· по сегменту номенклатуры;

· по товарной категории; по марке.

В левом окне формы подбора отображаются элементы выбранного пользователем вида подбора:список групп номенклатуры, список видов номенклатуры или список сегментов номенклатуры. список товарных категорий или список марок(брендов).
В правом окне формы показан список конкретных товаров, принадлежащих указанной в левой части группе

Подбор товара в документ производится в два этапа: сначала товар помещается в список отобранных товаров подбора, затем переносится в документ с помощью кнопки Перенести в документ. Товар помещается в список двойным щелчком по нужной строке. В процессе добавления товара реализована возможность вводить количество и цену, если пользователь нажмет кнопку Запрашивать количество и цену.

Если необходимо в списке подбора видеть только товары, которые есть в наличии на указанном в документе складе, установите флаг Только в наличии. Если необходимо показывать в подборе только те товары, по которым назначены цены, установите флаг Только с ценами. Для просмотра остатков на других складах и ожидаемых поступлениях используйте кнопку Остатки.
Для просмотра других цен по товару используйте кнопку Цены. При подборе товаров, для которых определен список дополнительных свойств (характеристик, дополнительных сведений, дополнительных реквизитов) можно использовать возможность дополнительного отбора по свойствам. Для реализации такого отбора необходимо установить вариант отбора По виду номенклатуры. В появившемся диалоговом окне будет показан список всех зарегистрированных свойств для выбранного вида номенклатуры. Для отбора нужного товара нужно установить флаги для нужных значений свойств и нажать на кнопку Применить.

Если учет номенклатуры ведется в разрезе характеристик, то после выбора такой номенклатуры программа предложит выбрать и конкретную характеристику. В случае ведения учета номенклатуры в разрезе характеристик выбор номенклатуры без характеристики невозможен.

Полнотекстовый поиск при подборе товаров

При установке флага Использовать полнотекстовый поиск при подборе (Настройки номенклатуры – Администрирование) область поиска расширяется на дополнительные реквизиты и свойства номенклатуры, характеристики номенклатуры и штрихкоды.
Осуществляется поддержка транслитерации (написание русских слов символами латиницы в соответствии с ГОСТом); поддержка замещения (написание части символов в русских словах одноклавишными латинскими символами); нечеткий поиск (буквы в найденных словах могут отличаться); поиск по одному и нескольким словам. В строке ввода допускается использование поисковых операторов типа И, ИЛИ, НЕ и т.д

Горячие клавиши

При подборе товаров можно использовать следующие горячие клавиши.
Alt+1 – переход к строке поиска.
Alt+Ctrl+1 – установить вариант отбора по иерархии номенклатуры.
Alt+Ctrl+2 – установить вариант отбора по виду номенклатуры.
Alt+Ctrl+3 – установить вариант отбора по сегменту номенклатуры.
Alt+2 – перейти к отбору.
Alt+3 – перейти к списку товаров.
Alt+4 – перейти к списку подобранных товаров.
Переходы также можно осуществлять с помощью клавиши Tab.
