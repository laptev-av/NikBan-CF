Настройка распределения товаров

Правила распределения поступления товаров пользователь может применить при распределении товаров из документов Поступление товаров и Перемещение товаров. Правила распределения продаж товаров пользователь может применить при распределении товаров из документов Перемещение товаров и Реализация товаров. Кроме того, правила распределения продаж товаров и услуг применяются при формировании документов продаж.

Настроить правила распределения можно с помощью сервисов, которые запускаются по ссылкам Распределение поступлений, Распределение продаж и Распределение продаж услуг из панели навигации карточки магазина.

Распределение товара при поступлении в магазин

Все товары, поступающие в магазин, по умолчанию приходуются на склад, который указан в поле Склад поступления в карточке данного магазина. С помощью схемы распределения поступлений по складам можно уточнить распределение в разрезе товарных групп товара.

В строках таблицы распределения показываются товарные группы товаров, а в столбцах – все склады магазинов, кроме указанного в качестве основного.
Чтобы отметить, на какой склад должен поступать товар данной товарной группы, нужно нажать клавишу Пробел или кнопку мыши в соответствующей ячейке таблицы.

Распределение товара при продаже

Если товар продается и отгружается при перемещении только с одного склада магазина, то настраивать распределение не имеет смысла. Этот склад должен быть указан в карточке магазина в поле Склад продажи. Он по умолчанию подставляется в документы Реализация товаров и Чек.

Распределение продаж можно настроить, чтобы уточнить склады продажи (отгрузки) для отдельных товарных групп. Таблицу распределения продаж можно заполнить из таблицы распределения поступлений, нажав кнопку Заполнить по распределению поступлений в командной панели. Изменить распределение можно, снимая или проставляя галочки в нужных ячейках нажатием кнопки мыши или клавиши Пробел.

Пример.Если весь поступающий в магазин товар распределяется по отделам магазина, из которых производится его продажа, то настройка распределения продаж совпадает с настройкой распределения поступления. Если же товар отдельных товарных групп проходит предпродажную подготовку в выделенных помещениях, то в распределении поступления для таких товарных групп указываются склады предпродажной подготовки, а в распределении продаж - отделы торгового зала. Перемещение подготовленных к продаже товаров производится с помощью документа Ордер на перемещение.

Можно также назначить рабочее место для оформления продаж товаров определенной товарной группы с определенного склада. В этом случае остатки товара по продажам с этого рабочего места будут списаны с указанного склада, в то время как на любом другом рабочем месте продажи будут производиться с общего склада продажи или со склада, указанного в распределении для товарной группы.

Пример.Сигареты и зажигалки размещены в прикассовой зоне всех касс магазина, три из которых находятся на первом этаже в отделе продажи продуктов питания, а «Касса 4» – в отделе хозтоваров в полуподвальном помещении. В этом случае для товарной группы «Сигареты», включающей в себя сигареты и зажигалки, устанавливается склад продажи «Продукты», а для рабочего места «Касса 4» – склад «Хозтовары».

Распределение продаж услуг по организациям

При создании магазина указывается организация, которая привязывается к складу продаж и по умолчанию подставляется в документы продаж. Если магазин продает услуги, которые оказывают другие организации, то нужно настроить таблицу Распределение продаж услуг. Настройка производится аналогично настройке таблиц распределения товаров, только в столбцах указаны не склады, а организации.
