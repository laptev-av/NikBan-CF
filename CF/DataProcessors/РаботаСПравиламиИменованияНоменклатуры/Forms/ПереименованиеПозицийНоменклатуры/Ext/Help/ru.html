Переименование позиций номенклатуры

Обработка для переименования позиций Номенклатуры по правилам. Обработка открывается по ссылке НСИ – Сервис - Переименование позиций номенклатуры и доступна если установлен флаг Правила именования номенклатуры по ссылке Администрирование – Настройки номенклатуры – Общие параметры.

С помощью обработки ответственный пользователь может заполнить недостающие параметры именования для позиций номенклатуры. Система анализирует карточки номенклатуры и предлагает пользователю значения незаполненных параметров, а также новые значения наименований номенклатуры.

К обработке доступны такие виды номенклатуры, по которым существует запись в регистре сведений Номенклатура к переименованию и правила именования которых не содержат ошибок. Запись в регистр сведений Номенклатура к переименованию происходит при установке правила именования в вид номенклатуры, а также при внесении изменений в используемые в видах номенклатуры правила именования. Справа от поля Вид номенклатуры отображается информация о количестве видов номенклатуры, которые требуется обработать.

Заполнение таблицы номенклатуры

Пользователь заполняет поле Вид номенклатуры. Справа от поля отображается информация о количестве позиций номенклатуры, которые необходимо переименовать.
Заполнение таблицы номенклатуры происходит при заполнении вида номенклатуры, либо по кнопке Получить значения из базы. В таблицу номенклатуры попадают те позиции вида номенклатуры, наименование которых сформировано не по правилу.

Таблица номенклатуры состоит из:

· Номенклатуры;

· Наименования по правилу;

· Параметров именования, которые генерируются в зависимости от параметров правила именования.

Необходимо заполнить те параметры, которые выделены текстом <Заполнить>. Синим цветом выделены значения, которые отличаются от значений в карточке номенклатуры и в дальнейшем подлежат записи в карточку номенклатуры. Чтобы по номенклатуре конкретной строки сформировалось наименование по правилу, необходимо заполнить все параметры именования в этой строке.

Для быстрого заполнения параметров именования значениями можно воспользоваться копированием и вставкой. Для копирования пользователю необходимо встать в нужную заполненную ячейку правой кнопкой мыши и в выпадающем меню выбрать действие Копировать, либо встать в нужную заполненную ячейку правой кнопкой мыши и воспользоваться сочетанием клавиш CTRL+C (Английская). Для вставки значения в строки пользователю необходимо выделить нужные ячейки в рамках текущей колонки (!), нажать на правую кнопку мыши в текущей колонке и в выпадающем меню выбрать действие Вставить, либо после выделения строк в рамках текущей колонки (!) воспользоваться сочетанием клавиш CTRL+V.

Если в процессе заполнения параметров в некоторых строках получится одинаковое наименование по правилу, то такие строки будут выделены цветом и будет отображена информация о количестве таких позиций номенклатуры.

Запись значений в базу

 

Перед записью значений в базу пользователю необходимо выделить строки таблицы номенклатуры, которые нужно записать. Для этого можно зажать клавишу CTRL, затем щелчком левой кнопки мыши добавить к выделению нужные строки. Для выделения всех строк можно воспользоваться сочетанием клавиш CTRL+A (Английская). После выделения строк пользователь нажимает на кнопку Записать значения в базу. Будут обработаны только те строки, в которых заполнено наименование по правилу. По каждой выделенной строке в позицию номенклатуры будут записаны значения, которые выделены синим цветом. После записи обработанные строки будут удалены из таблицы, также будет выдано оповещение о количестве переименованных позиций. Если после записи значений в таблице номенклатуры не останется строк, то данный вид номенклатуры будет считаться обработанным и соответствующая запись из регистра сведений Номенклатура к переименованию будет удалена.
