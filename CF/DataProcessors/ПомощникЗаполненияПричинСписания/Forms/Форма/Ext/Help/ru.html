Помощник заполнения причин списания

Сервис Помощник заполнения причин списания предназначен для подбора причин списания и создания документов Списание товаров в результате проведенной инвентаризации.

Сервис можно запустить из меню Сервис документа Приказ на проведение пересчета.

Для корректной работы помощника заполнения причин списания Пользователь должен сначала оформить результаты пересчета товаров на основании исходного приказа: провести документы Пересчет товаров и, в случае выполнения инвентаризации по ордерной схеме, провести ордера на отражение результатов пересчета. Если при пересчете зафиксированы недостачи, и эти недостачи не закрыты документами Пересортица товаров, то для каждой позиции с недостачей можно подобрать причины списания.

Для регистрации причин списания используется справочник Аналитики хозяйственных операций, в котором для хозяйственной операции Списание по инвентаризации могут быть созданы несколько аналитик. Справочник Аналитики хозяйственных операций заполняет администратор на странице Запасы и закупки раздела Настройка и администрирование.

Работа с помощником заполнения причин списания

В поле Аналитика хозяйственной операции в шапке формы Пользователь может выбрать значение аналитики из списка значений, которые соответствуют хозяйственной операции Списание по инвентаризации.

Выбранную аналитику можно записать во все строки табличной части, где аналитика ещё не заполнена, с помощью кнопки Заполнить пустые строки. Очистить значения аналитик во всех строках табличной части можно с помощью кнопки Очистить все строки. Кроме того, значение аналитики можно изменить отдельно в каждой строке.

Работа помощника может быть завершена с помощью кнопки Создать документы. В результате будет создано столько документов Списание товаров, сколько различных значений аналитик Пользователь выбрал для списываемых товаров в процессе работы с помощником. В каждом созданном документе табличная часть автоматически заполняется товарными позициями с соответствующими причинами списания.
