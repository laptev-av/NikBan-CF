Подбор по товарным категориям

Подбор по товарным категориям - единый для всех документов механизм подбора из Справочника номенклатуры

Подбор позиций товара производится с помощью команды Подобрать товары в командной панели табличной части документа. Пользователь может установить вариант подбора:

· по иерархии номенклатуры;

· по виду номенклатуры;

· по сегменту номенклатуры;

· по товарной категории;

· по марке.

В левом окне формы подбора отображаются элементы выбранного пользователем вида подбора.
В правом окне формы отображается номенклатура, входящая в выбранный элемент по указанному виду отбора.

Подбор товара в документ производится в два этапа: сначала товар помещается в список отобранных товаров подбора, затем переносится в документ с помощью кнопки Перенести в документ. Товар помещается в список двойным щелчком по нужной строке. В процессе добавления товара система предоставляет возможность вводить количество и цену, если пользователь нажмет кнопку Запрашивать количество и цену.

Возможности полнотекстового поиска при подборе товаров

При установке флага Использовать полнотекстовый поиск при подборе (Настройки номенклатуры – Администрирование) область поиска расширяется на дополнительные реквизиты и свойства номенклатуры, характеристики номенклатуры и штрихкоды.
Осуществляется поддержка транслитерации (написание русских слов символами латиницы в соответствии с ГОСТом); поддержка замещения (написание части символов в русских словах одноклавишными латинскими символами); нечеткий поиск (буквы в найденных словах могут отличаться); поиск по одному и нескольким словам. В строке ввода допускается использование поисковых операторов типа И, ИЛИ, НЕ и т. д
