Обработка "Помощник начала управления ассортиментом".

Обработка позволяет быстро сформировать текущий ассортимент форматов магазинов по заданным пользователем правилам.

На первой странице помощника пользователю необходимо установить формат для каждого магазина, на второй с помощью сервиса Заполнить по складским остаткам и оборотам сформировать документ ввода в ассортимент актуальных товарных позиций.
