Обработка "Контроль применения цен номенклатуры".

Для формирования документов Применение цен удобно использовать сервис Контроль применения цен (Маркетинг – Сервис). Пользователь, вводя параметры Магазин и Дата применение цен, может сформировать список товаров, на которые планируется изменение цен на выбранную дату. Причем на список может быть наложен произвольный настроенный пользователем отбор.

В заголовке табличной части Товары отображается количество строк. Это удобно для планирования объема работ по замене ценников в торговом зале. По команде Применить цены будет сформирован проведенный документ Применение цен на указанную в параметрах дату.

Контролировать правильность применения цен можно также с помощью отчета Цены к применению (Маркетинг – Отчеты – Применение цен). С помощью параметра Разница в днях можно вывести товары с просроченным на указанное количество дней применением цен. Отчет Результаты применения цен выводит товары, на которые за отчетный период цены были применены ошибочно: с просрочкой относительно даты назначения или с отсутствующим (например, удаленным) документом-основанием.
