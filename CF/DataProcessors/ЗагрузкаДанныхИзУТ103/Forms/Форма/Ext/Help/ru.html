Обработка "Загрузка данных из УТ 10.3"

С помощью этой обработки можно загрузить данные из конфигурации Управление торговлей, редакция 10.3. В форме обработки следует указать файл обмена, в который были выгружены данные из конфигурации Управление торговлей, редакция 10.3, и нажать на кнопку Загрузить.

Важно. Перенос данных из конфигурации Управление торговлей, редакция 10.3 возможен, начиная с версии 10.3.27.

Файлы, необходимые для выгрузки данных из конфигурации Управление торговлей, редакция 10.3 входят в поставку программы Розница, редакция 2.1 и находятся в каталоге Extfiles/ConvTr103.

  • Перенос данных УТ 10.3 - РТ 2.1.epf - внешняя обработка выгрузки данных во внешний файл из конфигурации Управление торговлей, редакции 10.3.

Порядок действий при переносе данных

Выполните резервное копирование информационной базы конфигурации "Управление торговлей" редакции 10.3. Если версия конфигурации "Управление торговлей" ниже 10.3.27, то проведите обновление конфигурации до указанной или более поздней версии. Выполните все регламентные процедуры в конфигурации "Управление торговлей", редакция 10.3.

  • Удалите помеченные на удаление объекты.
  • Закройте кассовые смены.
  • Оформите документы "Отчет комиссионера о продажах" и "Отчет комитенту о продажах".
  • Оформите межфирменные продажи между организациями.
  • Выполните проведение документов по партиям.

Важно. На момент переноса данных в информационной базе "Управление торговлей", редакция 10.3 не должно быть отрицательных остатков товаров по складам и организациям. Информация об отрицательных остатках товаров не будет выгружена из информационной базы конфигурации.

  • Откройте внешнюю обработку "Перенос данных УТ 10.3 - РТ 2.1.epf" с помощью пункта меню "Файл"-"Открыть". Обработка входит в поставку программы Розница, редакция 2.1 и находятся в каталоге Extfiles/ConvTr103.
  • В появившемся диалоге укажите имя внешнего файла, в который будет производиться выгрузка данных. Укажите дату, на которую актуальны остатки товаров, денежных средств и взаиморасчетов.
  • Укажите магазин(склад), которым заполняться соответствующие реквизиты справочников "Кассы", "Кассы ККМ", а так же документы "Ввод остатков расчетов с поставщиками".
  • В диалоговом окне появится список объектов, которые могут быть выгружены. По умолчанию будет выгружена информация по настройкам параметров учета, справочникам и остаткам, рекомендуемым к выгрузке. При необходимости можно отключить выгрузку информации по любому из справочников или отключить выгрузку остатков по любому из регистров конфигурации.
  • После настройки правил выгрузки данных необходимо нажать на кнопку "Выгрузить". Данные будут выгружены во внешний файл.
  • Произведите установку конфигурации Розница 2.1.
  • Создайте новую информационную базу конфигурации Розница 2.1.
  • Для загрузки данных выберите раздел "Администрирование" - "Первоначальное заполнение ИБ" - "Загрузка данных из Управление торговлей, ред. 10.3". В открывшемся диалоговом окне укажите внешний файл, в который была произведена выгрузка, и нажмите на кнопку "Загрузить". Данные будут загружены в информационную базу из внешнего файла.
  • После проведения загрузки убедитесь, что данные загружены корректно. С помощью отчетов проверьте правильность выгрузки остатков в обеих базах.

Важно. При повторной загрузке остатков данные в документах ввода остатков, созданных конвертацией, перезаполняются заново.
