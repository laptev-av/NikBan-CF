Помощник  формирования формата записи кода магнитной карты.

 

Предназначен для автоматического заполнения информации о формате магнитной карты при считывании магнитной карты считывателем магнитной карты. Порядок действий при работе с помощником:

1 этап.

Прочитайте карту считывателем магнитных карт. Перейдите Далее.При переходе на второй этап данные дорожек будут отображены в помощнике.

2 этап.

Скопируйте код карты в поле Код карты. Перейдите Далее. При переходе на третий этап произойдет автоопределение формата записи.

3 этап.

При необходимости откорректируйте Префикс и Суффикс формата записи, воспользовавшись контекстным меню поля Данные нужной дорожки. Перейдите Далее.

4 этап.

Прочитайте карту считывателем магнитных карт. По заданному формату записи будет выполнено определение кода карты. Если определение прошло успешно(заданный код карты равен считанному значению), перейдите Далее. При переходе на следующий этап будет выполнено сохранение введенных данных о формате записи.

5 этап.

На форме помощника отображаются сохраненные данные. Нажмите Готово.
