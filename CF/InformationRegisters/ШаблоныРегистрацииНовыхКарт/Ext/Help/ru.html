Шаблоны регистрации новых дисконтных карт

В данном регистре хранятся сведения о шаблонах, используемых при создании новых информационных карт. Шаблоны регистрационных карт открываются по ссылке Администрирование – Подключаемое оборудование - Шаблоны регистрации новых дисконтных карт.

В шаблоне указывается Вид карты – штриховая или магнитная, длина кода и начало и конец диапазона. При считывании или вводе кода карты вручную производится поиск данного кода в указанном пользователем диапазоне. Если карта не найдена, пользователю будет предложено зарегистрировать новую карту.

Помимо параметров поиска карты в шаблоне указываются данные для регистрации карты: Вид дисконтной карты и, если необходимо, Группа, в которую должна входить дисконтная карта указанного диапазона. Реквизит Группа владельца карты, служит для помещения владельца по завершении первичного опроса.

Флажок Проводить опрос владельца при регистрации управляет дальнейшими действиями пользователя – проводить опрос немедленно или вводить данные позднее по заполненной анкете.

Поле Бонусная программа лояльности позволяет в шаблоне указать программы лояльности, которые могут быть выбраны или созданы.

С помощью команды Проверить пересечения можно проверить, нет ли пересечений с уже сформированными диапазонами. Рекомендуется резервировать диапазоны карт заранее, до их изготовления. Однако пересечение диапазонов карт не критично для их регистрации по коду: кассиру будет предложено выбрать вид регистрируемой дисконтной карты.

Если считанный код карты не принадлежит ни одному из введенных администратором диапазонов, то автоматическая регистрация такой карты на кассе будет невозможна. Однако ее можно ввести в систему непосредственно в справочнике дисконтных карт (НСИ (Нормативно-справочная информация) – Физические лица – Дисконтные карты) пользователем, имеющим доступ к данному справочнику.
