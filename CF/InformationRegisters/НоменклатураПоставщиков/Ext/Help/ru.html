Поставляемая номенклатура

Это список поставляемой поставщиками номенклатуры. При проведении документов Заказ поставщику, Поступление товаров, Заказ на перемещение, Перемещение товаров в списке поставляемой номенклатуры автоматически регистрируются товары из табличной части этих документов, а в поле Дата последнего поступления устанавливается дата этих документов.

Список регистра сведений открывается по ссылке Закупки - Работа с поставщиками - Поставляемая номенклатура. Список номенклатуры может быть отфильтрован как по поставщику, так и по магазину, в который осуществляется поставка. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные сведения, задавая поиск по значениям различных реквизитов.

Список номенклатуры поставщика может также обновляться автоматически с помощью регламентного задания Обновление поставляемой номенклатуры. Регламентные задания можно настроить на странице регламентных заданий, которая открывается по ссылке Администрирование - Обслуживание - Регламентные и фоновые задания.

Обновление списка поставляемой номенклатуры

Обновление списка может понадобиться для удаления устаревших данных. Например, в том случае, когда поставщик вывел из своего ассортимента часть товарных позиций. Обновление производится с помощью сервиса Обновление номенклатуры поставщиков, который запускается из командной панели списка поставляемой номенклатуры.

Поля Магазин и Поставщик по умолчанию заполняются данными из отбора в исходном списке. Пользователь должен выбрать период. При нажатии кнопки Обновить поставляемую номенклатуру в список будет добавлена номенклатура из документов Заказ поставщику, Поступление товаров, Заказ на перемещение и Перемещение товаров за указанный пользователем период. Товары, поступавшие ранее, будут удалены из списка.

Новую запись в регистр сведений можно добавить с помощью кнопки Создать в командной панели списка.
